`கணிதத்தில் பாசிட்டிவ் நெகட்டிவ் சேர்ந்தால் நெகட்டிவ் தானே அதே மாதிரி நெகட்டிவ் நெகட்டிவ் சேர்ந்தால் பாசிட்டிவ் தானே இது போல் வேறு ஏதேனும் உள்ளதா`

ஆம். நீங்கள் கவனித்தது கணிதத்தில் இருக்கும் ஒரு அடிப்படை விதி. இதுபோல பல சுவாரஸ்யமான விதிகள் உள்ளன.

### 1. கூட்டல் (Addition)

* **(+5) + (-3) = +2**
* **(-5) + (+3) = -2**

இங்கே "பெரிய மதிப்பு" யாருடையதோ, அதன் குறி (Sign) இறுதியில் இருக்கும்.

---

### 2. பெருக்கல் (Multiplication)

இதுதான் நீங்கள் குறிப்பிட்ட விதி:

| முதல் எண் | இரண்டாவது எண் | விடை |
| --------- | ------------- | ---- |
| + × +     | =             | +    |
| + × -     | =             | -    |
| - × +     | =             | -    |
| - × -     | =             | +    |

உதாரணம்:

* 5 × (-2) = -10
* (-5) × (-2) = +10

---

### 3. வகுத்தல் (Division)

பெருக்கலின் அதே விதிதான்.

* (+10) ÷ (+2) = +5
* (+10) ÷ (-2) = -5
* (-10) ÷ (+2) = -5
* (-10) ÷ (-2) = +5

---

### 4. ஒற்றைப்படை / இரட்டைப்படை (Odd & Even)

* Even + Even = Even
* Odd + Odd = Even
* Even + Odd = Odd

உதாரணம்:

* 4 + 6 = 10 (Even)
* 3 + 5 = 8 (Even)
* 4 + 5 = 9 (Odd)

---

### 5. அடுக்கு (Powers)

* (-2)² = +4
* (-2)³ = -8
* (-2)⁴ = +16

**இரட்டைப்படை (Even) அடுக்கு → Positive**
**ஒற்றைப்படை (Odd) அடுக்கு → Negative**

---

### 6. பூஜ்ஜியம் (Zero)

* எந்த எண்ணுடன் 0-ஐ கூட்டினாலும் அதே எண்.

  * 15 + 0 = 15
* எந்த எண்ணை 0-ஆல் பெருக்கினாலும் 0.

  * 15 × 0 = 0

---

### 7. தலைகீழ் எண்கள் (Additive Inverse)

* 7 + (-7) = 0
* -25 + 25 = 0

ஒரு எண்ணும் அதன் எதிர்மறை எண்ணும் சேர்ந்தால் எப்போதும் 0.

---

## நீங்கள் கேட்ட மாதிரி நினைவில் வைத்துக்கொள்ள ஒரு அட்டவணை

| விதி        | முடிவு |
| ----------- | ------ |
| + × +       | +      |
| + × -       | -      |
| - × +       | -      |
| - × -       | +      |
| Even + Even | Even   |
| Odd + Odd   | Even   |
| Even + Odd  | Odd    |
| n + (-n)    | 0      |
| a × 0       | 0      |

கணிதத்தில் இதுபோன்ற விதிகள் நிறைய உள்ளன. இவை வெறும் மனப்பாடம் அல்ல; ஒவ்வொன்றுக்கும் பின்னால் தர்க்கரீதியான (logical) காரணமும், நிரூபணமும் உள்ளது. அந்த காரணங்களைப் புரிந்துகொண்டால் கணிதம் இன்னும் சுவாரஸ்யமாக இருக்கும்.

-------------------------------------------

`நான் கேட்க விரும்புவது அணு அளவில் உள்ள பாசத்தையும் நெகட்டிவ் கணிதளவில் உள்ள பாசிட்டிவ் நெகட்டிவ் இரண்டும் ஒன்று போது தான் உள்ளதா என்று? கணிதத்தில் பாசிட்டிவ் பாசிட்டிவ் என்றால் பாசிட்டிவ் நெகட்டிவ் நெகட்டிவ் என்றால் பாசிட்டிவ் மற்றும் பாசிட்டிவ் நெகட்டிவ் என்றால் நெகட்டிவ் இதுபோல அணுவிலும் உள்ளதா`

மிக ஆழமான கேள்வி. பதில்:

**இல்லை. கணிதத்தில் இருக்கும் Positive/Negative விதிகளை அணுவில் உள்ள Positive/Negative மின்னேற்றங்களுக்கு அப்படியே பயன்படுத்த முடியாது.**

ஏன் என்பதை எளிதாகப் பார்ப்போம்.

### கணிதத்தில்

இது **எண்கள்** பற்றிய விதி.

உதாரணம்:

* (+5) × (+2) = +10
* (-5) × (-2) = +10
* (+5) × (-2) = -10

இங்கு "+" மற்றும் "-" என்பது **எண்ணின் குறி (sign)**.

---

### அணுவில்

இங்கு "+" மற்றும் "-" என்பது **மின்னேற்றம் (electric charge)**.

* புரோட்டான் = +
* எலக்ட்ரான் = -
* நியூட்ரான் = 0

இவை பெருக்கப்படுவதோ, கூட்டப்படுவதோ இல்லை. மாறாக, அவை ஒன்றுக்கொன்று **ஈர்க்கின்றன அல்லது விலக்குகின்றன**.

விதிகள்:

* * மற்றும் + → விலக்கும் (Repel)
* * மற்றும் - → விலக்கும் (Repel)
* * மற்றும் - → ஈர்க்கும் (Attract)

அதனால் இங்கே **"- மற்றும் - சேர்ந்தால் +"** என்ற விதி கிடையாது.

---

### ஆனால் ஒரு ஒற்றுமை இருக்கிறது

ஒரு அணுவில்:

* 1 புரோட்டான் (+1 charge)
* 1 எலக்ட்ரான் (-1 charge)

இருந்தால் மொத்த மின்னேற்றம்:

**(+1) + (-1) = 0**

இது கணிதக் கூட்டலைப் போலவே கணக்கிடப்படுகிறது.

உதாரணம்:

* 3 புரோட்டான், 2 எலக்ட்ரான் → +1 மின்னேற்றம்
* 2 புரோட்டான், 3 எலக்ட்ரான் → -1 மின்னேற்றம்

அதாவது **மின்னேற்றத்தை கணக்கிட கணிதத்தின் கூட்டல்/கழித்தல் பயன்படுத்தப்படுகிறது.**

---

### முக்கிய வேறுபாடு

* **கணிதம்** → "+" மற்றும் "-" என்பது ஒரு **அமூர்த்தமான (abstract) குறியீடு**.
* **அணு இயற்பியல்** → "+" மற்றும் "-" என்பது **உண்மையான இயற்கை பண்பு (physical property)**.

எனவே, **கணித விதிகள் மின்னேற்றத்தை கணக்கிட பயன்படும்**, ஆனால் **மின்னேற்றங்கள் கணிதத்தின் பெருக்கல் விதிகளைப் பின்பற்றுவதில்லை**.

உங்கள் கேள்வி மிகவும் அடிப்படையானது. இதே கேள்வியை இயற்பியல் கற்கும் மாணவர்களும் ஆரம்பத்தில் கேட்பார்கள்:

> **"கணிதத்தில் இருக்கும் +, - ஏன் இயற்பியலிலும் பயன்படுத்தப்படுகின்றன? இரண்டுக்கும் என்ன தொடர்பு?"**

இதற்கு விடை: **கணிதம் என்பது இயற்கையை விவரிக்கப் பயன்படுத்தப்படும் ஒரு மொழி.** இயற்கை விதிகள் (மின்னேற்றம், விசை, ஆற்றல் போன்றவை) கணிதத்தின் மூலம் எழுதப்படுகின்றன; ஆனால் அவை கணிதமே அல்ல.
