இந்த உதாரணத்தில் குழப்பம் வருவது இயல்பு. மெதுவாகப் பார்ப்போம்.

கொடுக்கப்பட்டுள்ளது:

(2+3x)^2

முதலில் square-ன் formula:

genui{"math_block_widget_always_prefetch_v2":{"content":"(a+b)^2=a^2+2ab+b^2"}}

இப்போது:

```text
a = 2
b = 3x
```

என்று வை.

அப்போது:

```text
(2+3x)^2
=
2² + 2(2)(3x) + (3x)²
```

---

### முதல் term

```text
2²
=
2×2
=
4
```

---

### நடுவில் இருக்கும் term

```text
2(2)(3x)
```

இங்கே முதல் `2` formula-லிருந்து வந்தது:

```text
2ab
```

இல் இருக்கும் `2`.

---

இப்போது:

```text
2×2×3x
=
12x
```

ஏனெனில்:

```text
2×2×3 = 12
```

---

### கடைசி term

```text
(3x)²
```

என்பது:

```text
(3x)(3x)
```

---

இப்போது:

```text
3×3×x×x
```

---

அதனால்:

```text
9x²
```

---

எனவே:

```text
(2+3x)²
=
4 + 12x + 9x²
```

---

நீ குழப்பப்படும் இடம் பொதுவாக இதுதான்:

```text
(3x)²
```

இது

❌

```text
3x²
```

அல்ல.

ஏனெனில் square முழு `3x`-க்கும் பொருந்துகிறது.

அதனால்:

```text
(3x)²
=
3²x²
=
9x²
```

---

Verify பண்ணிப் பார்ப்போம்.

`x = 1` என்று வைத்தால்:

LHS:

```text
(2+3×1)²
=
5²
=
25
```

RHS:

```text
4 + 12(1) + 9(1)²
=
4 + 12 + 9
=
25
```

இரண்டும் சமம்.

இதனால் formula சரி என்பதைப் பார்க்கலாம். 🎯

இங்கே உனக்கு அதிக குழப்பம் வருவது `2(2)(3x)`-லா அல்லது `(3x)²`-லா? அந்தப் பகுதியை மட்டும் இன்னும் ஆழமாக உடைத்து காட்டுகிறேன்.

------------------------------------

உன் முயற்சி நல்லது. ஆனால் ஒரு இடத்தில் தான் பிரச்சனை வந்துள்ளது.

புத்தகத்தில்:

```text
(-2a+5b)²
```

என்பதை

```text
a = -2a
b = 5b
```

என்று வைத்து formula பயன்படுத்துகிறார்கள்:

```text
(a+b)² = a² + 2ab + b²
```

அதனால்:

```text
(-2a+5b)²
=
(-2a)² + 2(-2a)(5b) + (5b)²
```

---

முதல் term:

```text
(-2a)²
=
4a²
```

---

நடுவில்:

```text
2(-2a)(5b)
```

முதலில்:

```text
(-2a)(5b)
=
-10ab
```

ஆகவே:

```text
2(-10ab)
=
-20ab
```

---

கடைசி term:

```text
(5b)²
=
25b²
```

---

எனவே:

```text
4a² - 20ab + 25b²
```

---

### உன் முயற்சியில் என்ன தவறு?

நீ இதை எழுதியிருக்கிறாய்:

```text
4a² + (-10ab) + (-10ab) + 25b²
```

இதுவரை சரி! ✅

ஏனெனில்:

```text
2(-10ab)
=
(-10ab)+(-10ab)
```

---

பிறகு:

```text
(-10ab)+(-10ab)
=
(-10 + -10)ab
=
-20ab
```

என்று நேரடியாக எழுத வேண்டும்.

---

ஆனால் நீ அடுத்தடுத்த வரிகளில்:

```text
(-1)+(-1)
```

மற்றும்

```text
20ab
```

என்று coefficient-ஐ பிரித்து மீண்டும் சேர்க்க முயற்சி செய்திருக்கிறாய்.

அங்கே sign manipulation கொஞ்சம் குழப்பமாகி விட்டது.

---

### மிக எளிய வழி

```text
(-10ab)+(-10ab)
```

↓

```text
(-10 + -10)ab
```

↓

```text
-20ab
```

அவ்வளவுதான்.

---

இன்னும் ஒரு super-simple check:

```text
ab = 1
```

என்று வைத்துக்கொள்.

அப்போது:

```text
(-10ab)+(-10ab)
=
-10 + -10
=
-20
```

---

அதனால்:

```text
(-10ab)+(-10ab)
=
-20ab
```

தான், **`-2(20ab)` அல்ல**.

அதுதான் உன் கணக்கில் வந்த முக்கிய தவறு. 🎯

உண்மையில் இந்த உதாரணத்தை புத்தகத்தின் formula பயன்படுத்தாமல், முழுவதும் expand செய்து:

```text
(-2a+5b)(-2a+5b)
```

என்று ஆரம்பித்து `4a² - 20ab + 25b²` எப்படி வருகிறது என்பதையும் காட்ட முடியும். அது இன்னும் தெளிவாக இருக்கும்.

-----------------------------------------------

ஒரு mathematician இந்த expression-ஐ:

```text
2x² + x - 4x - 2
```

என்று மட்டும் பார்க்க மாட்டார்.

அவர் முதலில் **terms**-ஐ பிரிப்பார்.

```text
2x²
+
x
+
(-4x)
+
(-2)
```

இப்படி.

நீ செய்தது போல:

```text
2x² + x + (-4x) - 2
```

என்று எழுதுவது நல்ல பழக்கம். 👍

---

பிறகு mathematician கேட்பார்:

> "Like terms எவை?"

அதாவது ஒரே மாதிரி terms எவை?

---

இங்கே:

```text
x
```

மற்றும்

```text
-4x
```

இரண்டும் x terms.

---

அதனால்:

```text
x + (-4x)
```

முதலில் சேர்க்கப்படும்.

---

Coefficient மட்டும் பார்ப்பார்:

```text
1x + (-4)x
```

↓

```text
(1 + (-4))x
```

↓

```text
-3x
```

---

அதனால்:

```text
2x² + x - 4x - 2
```

↓

```text
2x² - 3x - 2
```

---

### இன்னும் ஆழமான mathematician பார்வை

அவர் இதை இப்படி பார்ப்பார்:

```text
2x² + x - 4x - 2
```

↓

```text
2x² + (1-4)x - 2
```

↓

```text
2x² - 3x - 2
```

---

கவனி:

```text
x + (-4x)
```

என்பது

```text
(1 + (-4))x
```

ஆனது எப்படி?

இது distributive law-ன் reverse:

```text
ab + cb
=
(a+c)b
```

---

இங்கே:

```text
1x + (-4)x
=
(1+(-4))x
```

---

அதனால்:

```text
=
(-3)x
=
-3x
```

---

நீ இப்போது Algebra படிக்கும்போது ஒவ்வொரு expression-ஐயும் இப்படி பார்க்க ஆரம்பிக்க வேண்டும்:

```text
2x² + x - 4x - 2
```

↓

```text
2x² + (1x) + (-4x) + (-2)
```

↓

```text
2x² + (1+(-4))x - 2
```

↓

```text
2x² - 3x - 2
```

இதுதான் ஒரு mathematician-ன் natural mental process. 🎯

அதாவது அவர் `x-4x` என்று உடனே பார்க்க மாட்டார்; முதலில்

```text
1x + (-4)x
```

என்று coefficient-களைப் பிரித்து பார்ப்பார். அதுதான் algebraic thinking.


---------------------------------------------------

`Coefficient என்றால்`


**Coefficient** என்பது ஒரு variable-க்கு (மாறி) முன்னால் multiply ஆகி நிற்கும் எண்.

உதாரணம்:

```text
5x
```

இங்கே:

* Variable = `x`
* Coefficient = `5`

ஏனெனில்:

```text
5x = 5 × x
```

---

மற்ற உதாரணங்கள்:

```text
7a
```

Coefficient = `7`

---

```text
-3y
```

Coefficient = `-3`

---

```text
10ab
```

Coefficient = `10`

---

```text
-20ab
```

Coefficient = `-20`

---

இப்போது நீ கேட்ட expression:

```text
2x² + x - 4x - 2
```

இதில்:

| Term  | Coefficient                               |
| ----- | ----------------------------------------- |
| `2x²` | `2`                                       |
| `x`   | `1`                                       |
| `-4x` | `-4`                                      |
| `-2`  | Constant (coefficient பற்றி பேச மாட்டோம்) |

---

ஏன் `x`-க்கு coefficient `1`?

ஏனெனில்:

```text
x = 1x
```

---

அதனால்:

```text
x - 4x
```

என்பது உண்மையில்:

```text
1x + (-4)x
```

---

Distributive law reverse:

```text
(1 + (-4))x
```

↓

```text
-3x
```

---

ஒரு mathematician:

```text
x - 4x
```

பார்த்தவுடன் மனதில்:

```text
1x - 4x
```

என்று மாற்றி coefficient-களை மட்டும் சேர்ப்பார்:

```text
1 + (-4)
=
-3
```

அதனால்:

```text
x - 4x
=
-3x
```

🎯 **சுருக்கமாக:**

> **Coefficient = Variable-க்கு முன்னால் multiply ஆகி நிற்கும் எண்.**

உதாரணம்:

```text
8x²
```

இல் coefficient = `8`

```text
-15ab
```

இல் coefficient = `-15`

```text
x
```

இல் coefficient = `1` (மறைந்து இருக்கும் 1).
