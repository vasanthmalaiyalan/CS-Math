`real number, rational number என்றால் என்ன அது எப்படி அடையா ளம் கண்டு அடையாளம் கண்டு கொள்வது`

மிகவும் முக்கியமான கேள்வி. **Serge Lang** புத்தகத்தில் இது ஒரு பெரிய Idea.

பெரும்பாலானவர்கள் இப்படி நினைப்பார்கள்:

> Integers → Rational Numbers → Real Numbers

ஆனால் கணிதவியலாளர் இப்படி நினைப்பார்:

> **ஒவ்வொரு புதிய Number System-மும் முந்தையதை உள்ளடக்கிய (extends) ஒரு பெரிய தொகுப்பு.**

அதை முதலில் பார்ப்போம்.

---

# 1. Integer (ℤ)

இவை முழு எண்கள்.

```text
..., -3, -2, -1, 0, 1, 2, 3, ...
```

இதில் Fraction கிடையாது.

உதாரணம்:

```text
7
-5
0
102
```

அனைத்தும் Integer.

---

# 2. Rational Number (ℚ)

வரையறை (Definition):

ஒரு எண்ணை

```text
m/n
```

என்று எழுத முடிந்தால்,

இங்கு

* m = Integer
* n = Integer
* n ≠ 0

அந்த எண் Rational Number.

அதாவது

```text
1/2
3/7
-5/9
8/1
```

அனைத்தும் Rational.

# Real Number (ℝ)

இப்போது Real Number.

இதன் Definition வேறு.

**Number Line-ல் இருக்கும் ஒவ்வொரு புள்ளியும் ஒரு Real Number.**

அதாவது

```
<-----------●-----------●-----------●----------->
           -2          0           3
```

இந்த Line-ல் இருக்கும்

**எல்லா புள்ளிகளும்**

Real Numbers.

# Serge Lang-ன் பார்வை

அவர் Number Systems-ஐ இப்படிப் பார்க்கிறார்.

```text
Real Numbers (ℝ)
│
├── Rational Numbers (ℚ)
│      │
│      ├── Integers (ℤ)
│      │      │
│      │      ├── Whole Numbers
│      │      │      │
│      │      │      └── Natural Numbers
│
└── Irrational Numbers
```

---

## மிக எளிய அடையாள விதி

**Step 1:**

இந்த எண்ணை

```text
m/n
```

வடிவில் எழுத முடியுமா?

✅ முடிந்தால் → **Rational Number**

❌ முடியாவிட்டால் → அடுத்த கேள்வி.

---

**Step 2:**

இந்த எண் Number Line-ல் ஒரு இடத்தில் இருக்கிறதா?

✅ ஆம் → **Real Number (Irrational)**

---

### மனதில் வைத்துக்கொள்ள வேண்டிய ஒரு வரி

* **Rational Number = Fraction (m/n) ஆக எழுதக்கூடிய எண்.**
* **Real Number = Number Line-ல் இருக்கும் எந்த எண் வேண்டுமானாலும் (Rational + Irrational).**

இதுதான் கணிதவியலாளர்கள் பயன்படுத்தும் அடிப்படை பார்வை.
