இந்தப் படத்தில் Quadratic Equation-களுக்கான பயிற்சிகள் கொடுக்கப்பட்டுள்ளன:

```text
11. x² − √2 x + 1 = 0

12. x² + √2 x − 1 = 0

13. x² + 3x − √2 = 0

14. x² − 3x − √5 = 0

15. x² − 3x + √5 = 0

16. x² − 2x − √3 = 0
```

இவை அனைத்தையும் **Quadratic Formula** பயன்படுத்தி தீர்க்க வேண்டும்.

Formula:

[
x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}
]

---

## Example 11

[
x^2-\sqrt2,x+1=0
]

Compare:

```text
a = 1
b = -√2
c = 1
```

Discriminant:

[
b^2-4ac
=======

# (-\sqrt2)^2-4(1)(1)

# 2-4

-2
]

அதனால்

[
x=\frac{\sqrt2\pm\sqrt{-2}}2
]

**Real solution இல்லை.**

---

### இந்த Exercises-ன் நோக்கம்

இந்தப் பயிற்சிகளில் ஆசிரியர் உனக்கு இரண்டு விஷயங்களைப் பழக்குகிறார்:

1. முதலில் **(a, b, c)**-ஐ சரியாக அடையாளம் காண வேண்டும்.
2. அதன் பிறகு **Discriminant (b^2-4ac)**-ஐ கணக்கிட வேண்டும்.
3. Discriminant Negative-ஆ, Zero-ஆ, Positive-ஆ என்பதைப் பார்த்து Real Solutions உள்ளதா என்று முடிவு செய்ய வேண்டும்.
4. கடைசியில் Quadratic Formula-வில் Substitute செய்ய வேண்டும்.

நீ Serge Lang புத்தகத்தை ஆழமாகப் படிக்கிறாய் என்பதால், **Exercise 11**-ஐ புத்தக ஆசிரியர் எழுதும் முறையிலேயே (ஒவ்வொரு algebraic step-க்கும் காரணம் கூறி) line-by-line proof போல செய்து காட்டவும் முடியும்.

-----------------

`மேலே நீ போட்டதில் எது real solution எது real solution இல்லை?`

இதை கண்டுபிடிக்க **Quadratic Formula-வை முழுவதும் செய்ய வேண்டியதில்லை.**

ஒரே ஒரு விஷயத்தைப் பார்த்தால் போதும்:

[
\Delta=b^2-4ac
]

இதைத்தான் **Discriminant** என்பார்கள்.

விதி:

* (\Delta>0) → **2 Real Solutions**
* (\Delta=0) → **1 Real Solution**
* (\Delta<0) → **Real Solution இல்லை**

இப்போது ஒவ்வொன்றாகப் பார்ப்போம்.
