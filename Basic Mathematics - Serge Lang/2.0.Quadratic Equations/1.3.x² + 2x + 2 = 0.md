இந்த Example 2-ல் ஆசிரியர் **ஒரு Quadratic Equation-க்கு Real Number solution இல்லாத சூழ்நிலையை** காட்டுகிறார்.

மெதுவாகப் பார்ப்போம்.

---

# Step 1

Equation:

```text
x² + 2x + 2 = 0
```

முதலில்

`+2`-ஐ வலது பக்கம் அனுப்புகிறார்.

```text
x² + 2x = -2
```

இதுவரை நமக்குத் தெரிந்தது.

---

# Step 2

இப்போது அவர் மீண்டும் **Completing the Square** செய்கிறார்.

Formula:

(x+1)^2=x^2+2x+1

அவர் பார்க்கிறார்.

நம்மிடம்

```text
x² + 2x
```

இருக்கிறது.

Formula-வில்

```text
x² + 2x + 1
```

இருக்கிறது.

அதனால்

இரு பக்கங்களிலும் `+1` சேர்க்கிறார்.

```text
x² + 2x + 1
=
-2 + 1
```

---

# Step 3

இப்போது இடது பக்கம் Formula ஆக மாறுகிறது.

```text
(x+1)²
=
-1
```

இதுவரை எல்லாம் சரி.

---

# Step 4 (இதுதான் முக்கியமான Point)

இப்போது ஆசிரியர் கேட்கிறார்.

> **"(x+1)² = -1 சாத்தியமா?"**

இதை சோதிப்போம்.

---

வைத்துக்கொள்

```text
x = 5
```

அப்படியானால்

```text
(x+1)²
=
6²
=
36
```

Positive.

---

வைத்துக்கொள்

```text
x = -3
```

அப்படியானால்

```text
(x+1)²
=
(-2)²
=
4
```

Positive.

---

வைத்துக்கொள்

```text
x = -1
```

அப்படியானால்

```text
(x+1)²
=
0²
=
0
```

Zero.

---

**எந்த Real Number எடுத்தாலும்**

Square செய்தால்

```text
Positive
```

அல்லது

```text
Zero
```

மட்டுமே வரும்.

---

## ஆனால்

Equation என்ன சொல்கிறது?

```text
(x+1)²
=
-1
```

அதாவது

ஒரு Square

Negative ஆக வேண்டும் என்று கேட்கிறது.

---

அது முடியாது.

ஏன்?

ஏனெனில்

```text
(+ எண்)² = Positive
```

```text
(- எண்)² = Positive
```

```text
0² = 0
```

**Negative ஒருபோதும் வராது.**

---

# அதனால்தான் ஆசிரியர் எழுதுகிறார்

> **But a negative real number cannot possibly be a square of a real number.**

அதாவது

> **ஒரு Real Number-ன் Square ஒருபோதும் Negative ஆக முடியாது.**

---

# அதனால் முடிவு

```text
(x+1)² = -1
```

என்பதை பூர்த்தி செய்யும்

**Real Number x எதுவும் இல்லை.**

அதனால்

இந்த Equation-க்கு

**Real Number Solution இல்லை.**

---

## ஆனால் ஒரு முக்கியமான விஷயம்

இந்த புத்தகத்தில் இப்போது **Real Numbers** மட்டுமே படிக்கிறோம்.

பிறகு **Complex Numbers** Chapter-க்கு சென்றால்,

```text
x² = -1
```

என்பதற்கு

```text
x = i
```

என்ற Solution கிடைக்கும்.

ஆனால் **இந்த Example-ல் ஆசிரியர் வேண்டுமென்றே Real Numbers-க்குள் மட்டுமே பேசுகிறார்.**

---

### இந்த Example-ன் ஒரே நோக்கம்

ஆசிரியர் சொல்ல வருவது:

> **"எல்லா Quadratic Equations-க்கும் Real Number solution கிடைக்காது. சில சமயம் Completing the Square செய்த பிறகு `(something)² = negative` வந்தால், அங்கேயே Real Number-ல் solution இல்லை என்று முடிவு செய்யலாம்."**
