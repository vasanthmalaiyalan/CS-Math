இந்த Exercise 5-ல் உள்ள எல்லா equations-க்கும் ஒரே principle:

> `x`-ஐ தனியாக விடு.

நீ முதலில் முயற்சி செய்து பார்க்கலாம். Answer மட்டும் கீழே:

```text
a)  2x - 7 = 21
    x = 14

b)  3(2x - 5) = 7
    x = 11/3

c)  (4x - 1)2 = 1/4
    x = 3/8

d)  -4x + 3 = 5x
    x = 1/3

e)  3x - 2 = -5x + 8
    x = 5/4

f)  3x + 2 = -3x + 4
    x = 1/3

g)  (4x)/3 + 1 = 3x
    x = 3/5

h)  -(3x)/2 + 4/3 = 5x
    x = 8/39

i)  (2x - 1)/3 + 4x = 10
    x = 31/14
```

⚠️ ஒரு முக்கிய குறிப்பு:

```text
(4x - 1)² = 1/4
```

போல தெரிகிறது.

அது உண்மையிலேயே square (`²`) என்றால் **இரண்டு solutions** வரும்:

```text
4x - 1 = 1/2
```

அல்லது

```text

```

அதனால்:


```text
x = 1/8
```

ஆனால் புத்தகத்தின் இந்த chapter level-ஐ பார்த்தால் அது `(4x−1)/2 = 1/4` ஆகவும் இருக்க வாய்ப்பு இருக்கிறது. படத்தில் square சின்னம் தெளிவாக தெரியவில்லை.

(c)-க்கு மட்டும் image-ஐ zoom செய்து அல்லது அந்த expression-ஐ text-ஆக type பண்ணினால் சரியான solution-ஐ step-by-step காட்டுகிறேன்.

``2x - 7 = 21`இதுல மைனர் சேலை கேன்சல் செய்வதற்கு பிளஸ் 7 பயன்படுத்துகிறோம் ஆனால் அந்த 2x என்று இருக்கும் அந்த வேல்யூ multiplication குறிக்கக் கூடியது ஆனால் அதை கேன்சல் செய்வதற்கு எதற்காக divideபயன்படுத்த வேண்டும்?`

இதுதான் Algebra-வின் மிக முக்கியமான idea. நீ சரியான கேள்வியை கேட்டிருக்கிறாய்.

நாம்:

```text
2x - 7 = 21
```

என்று வைத்துக்கொள்வோம்.

---

முதலில்:

```text
-7
```

ஐ நீக்கினோம்.

ஏன்?

ஏனெனில் `-7` என்பது **addition operation**-ல் வந்திருக்கிறது.

```text
2x + (-7)
```

என்று உள்ளது.

---

Addition-ஐ cancel செய்ய:

```text
+7
```

சேர்க்கிறோம்.

ஏனெனில்:

```text
(-7) + 7 = 0
```

இது additive inverse.

---

அதனால்:

```text
2x - 7 = 21
```

↓

```text
2x = 28
```

---

இப்போது:

```text
2x
```

என்பது என்ன?

```text
2 × x
```

---

இங்கே operation addition இல்லை.

இங்கே operation:

```text
multiplication
```

---

அதனால் additive inverse பயன்படுத்த முடியாது.

---

இப்போது கேள்வி:

> 2 × x = 28

இதில் 2-ஐ எப்படி "நீக்குவது"?

---

Multiplication-ஐ cancel செய்ய அதன் inverse தேவை.

Addition-க்கு inverse:

```text
-a
```

---

Multiplication-க்கு inverse:

```text
1/a
```

---

2-ன் multiplicative inverse:

```text
1/2
```

---

அதனால் இருபக்கத்தையும்:

```text
1/2
```

ஆல் பெருக்குகிறோம்.

```text
(1/2)(2x)
=
(1/2)28
```

---

இடது பக்கம்:

```text
(1/2)×2 = 1
```

---

ஆக:

```text
x = 14
```

---

நாம் பள்ளியில் இதை:

```text
28 ÷ 2
```

என்று எழுதுகிறோம்.

ஆனால் Algebra பார்வையில் உண்மையில் நடப்பது:

```text
(1/2) × 28
```

---

அதனால்தான்:

```text
3x = 8
```

என்றால்

```text
x = 8/3
```

---

ஏனெனில்:

```text
(1/3)(3x)
=
x
```

---

இதை நீ ஏற்கனவே பார்த்த இரண்டு inverse-களோடு ஒப்பிடு:

### Addition

```text
x + 5 = 10
```

↓

```text
x + 5 + (-5)
=
10 + (-5)
```

↓

```text
x = 5
```

---

### Multiplication

```text
5x = 10
```

↓

```text
(1/5)(5x)
=
(1/5)10
```

↓

```text
x = 2
```

---

அதனால்:

* `+5`-ஐ நீக்க `-5` பயன்படுத்துகிறோம்.
* `×5`-ஐ நீக்க `1/5` பயன்படுத்துகிறோம்.

இதுதான் Serge Lang இந்த முழு chapter-ல் உருவாக்கிக் கொண்டிருக்கும் பெரிய idea:

```text
-a
```

= additive inverse

```text
1/a
```

= multiplicative inverse

மற்றும் equation solve செய்வது என்பது உண்மையில் **சரியான inverse-ஐ பயன்படுத்தி operation-ஐ undo செய்வது** தான்.

`(4x - 1) 2 = 1/4 இதுதான் புத்தகத்தில் இருப்பது `

நீ எழுதியது:

```text
(4x - 1) 2 = 1/4
```

இங்கே ஒரு ambiguity இருக்கிறது.

கணித notation-ல்:

```text
(4x - 1)2
```

என்றால் சாதாரணமாக

```text
(4x - 1) × 2
```

என்றுதான் அர்த்தம்.

ஏனெனில் square எழுதினால்:

```text
(4x - 1)^2
```

அல்லது

```text
(4x - 1)²
```

என்று மேலே சிறிய 2 இருக்கும்.

---

நீ செய்த முதல் step:

```text
(4x - 1)2
↓
8x - 2
```

இது சரி.

Distributive law:

```text
2(4x - 1)
=
2·4x - 2·1
=
8x - 2
```

---

ஆனால் அடுத்த step-ல் தவறு.

நீ:

```text
8x - 2 = 1/4
```

இருந்து

```text
8x = 2/4
```

என்று போயிருக்கிறாய்.

அது தவறு.

ஏனெனில்:

```text
8x = 1/4 + 2
```

---

இப்போது:

```text
2 = 8/4
```

அதனால்:

```text
1/4 + 2
=
1/4 + 8/4
=
9/4
```

---

ஆக:

```text
8x = 9/4
```

---

இருபக்கத்தையும் 8-ஆல் divide செய்:

```text
x = (9/4)/8
```

---

```text
x = 9/(4×8)
```

---

```text
x = 9/32
```

---

Check:

```text
2(4×9/32 - 1)
```

```text
= 2(36/32 - 1)
```

```text
= 2(9/8 - 1)
```

```text
= 2(1/8)
```

```text
= 1/4
```

✔ சரியாக வருகிறது.

அதனால் **உன் algebra method சரி**, ஆனால்

```text
1/4 + 2
```

என்ற fraction addition-ல் தவறு நடந்ததால் answer தவறிவிட்டது.

சரியான answer:

```text
x = 9/32
```

ஆனால் புத்தகத்தில் உண்மையில் `²` (square) இருந்தால் answer வேறு வரும். எனவே அந்த ஒரு expression-ஐ மட்டும் தெளிவாக type பண்ணு அல்லது crop செய்து அனுப்பு:

```text
(4x - 1)2 = 1/4
```

இல் உள்ள `2` சாதாரண 2-ஆ, அல்லது மேலே இருக்கும் square `²`-ஆ? அதுதான் முடிவை தீர்மானிக்கும்.
