# முதலில்: Real Number என்றால் என்ன?

Serge Lang எழுதுகிறார்:

> Integers மற்றும் Rational Numbers என்பது Real Numbers-ன் ஒரு பகுதி மட்டுமே.

அதாவது,

```
Real Numbers
│
├── Integers
│      -3 -2 -1 0 1 2 3
│
├── Rational Numbers
│      1/2
│      -7/5
│      13/9
│
└── Irrational Numbers
       √2
       π
       e
```

இதுவரை நாம்

* Integer
* Rational Number

மட்டும் படித்தோம்.

இப்போது

**√2, π, e** போன்றவற்றையும் சேர்த்து Real Numbers படிக்கப் போகிறோம்.

# Real Number எப்படி வரையறுக்கிறார்?

Serge Lang இரண்டு விதமாக சொல்கிறார்.

## முதல் Definition

Number line-ல் இருக்கும் **ஒவ்வொரு புள்ளியும்** ஒரு Real Number.

உதாரணம்

```
------|------|------|------|------
     -2     -1      0      1      2
```

இது மட்டும் இல்லை.

0 மற்றும் 1 இடையே

```
0.5
0.75
0.9132415...
```

எல்லாமே Real Numbers.

---

## இரண்டாவது Definition

Infinite decimal expansion இருந்தாலும் அது Real Number.

உதாரணம்

```
9.123145....
```

இதுவும் Real Number.

# Property 1

## Commutative Law of Addition

```
a+b=b+a
```

Meaning:

```

```

```
8=8
```

Order மாறினாலும் answer மாறாது.

# Property 2

## Associative Law

```

```

```

=
9
```

Answer மாறவில்லை.

# Property 3

## Zero Identity

```

```

Example

```

```

Zero சேர்த்தாலும் number மாறாது.

# Property 4

## Additive Inverse

ஒவ்வொரு எண்ணுக்கும்

ஒரு opposite number இருக்கும்.

```

```

```

```


இதைத்தான்

**Additive Inverse**

என்று சொல்கிறார்.

# இந்த Section-ன் முழு சுருக்கம்

Serge Lang புதிய கணிதம் கற்றுக்கொடுக்கவில்லை. அவர் சொல்வது:

1. **Real Numbers** என்பது Integer + Rational + Irrational ஆகியவற்றை உள்ளடக்கிய பெரிய தொகுப்பு.
2. Integer மற்றும் Rational Number-க்கு கற்றுக்கொண்ட Addition, Multiplication, Inverse போன்ற அனைத்து அடிப்படை algebra விதிகளும் Real Numbers-க்கும் அப்படியே பொருந்தும்.
3. இதுவே அடுத்த chapters-ல் Calculus, Functions, Geometry போன்ற தலைப்புகளுக்கான அடித்தளமாக இருக்கும்.

என் பரிந்துரை: **இந்த section-ல் உள்ள "Uniqueness of Additive Inverse" மற்றும் "Uniqueness of Multiplicative Inverse" proofs**-ஐ அடுத்ததாக தனித்தனியாக, ஏன் ஒவ்வொரு வரியும் சரி என்று மிக ஆழமாகப் புரிந்துகொள்வது. அந்த இரண்டு proofs-தான் இந்த section-ன் உண்மையான மையம்.
