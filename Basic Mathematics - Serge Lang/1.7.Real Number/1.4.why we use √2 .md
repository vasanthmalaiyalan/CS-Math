`முதலில் இதை நாம் எதற்காக தெரிந்து கொள்கிறோம் மற்றும் என் மேலே சொன்னதெல்லாம் புரிந்தது ஆனால் √2 இதை நாம் எதற்காக போடுகிறோம்? `

இந்தக் கேள்விதான் மிகவும் முக்கியமானது. 👍

நீ கேட்பது:

> **"√2-வை எதற்காக அறிமுகப்படுத்துகிறோம்?"**

இது வெறும் notation (குறியீடு) அல்ல. **கணிதத்தில் ஒரு பெரிய பிரச்சினையைத் தீர்க்கவே இதை உருவாக்கினார்கள்.**

# இப்போது முக்கியமான கேள்வி

```text
x² = 2
```

Answer என்ன?

2-ஐ square செய்தால்?

```text
2² = 4
```

1-ஐ square செய்தால்?

```text
1² = 1
```

அப்படியானால்

**1-க்கும் 2-க்கும் நடுவில் ஏதோ ஒரு எண் இருக்க வேண்டும்.**

அதன் square மட்டும் 2 ஆக வேண்டும்.

அந்த எண்ணுக்கு என்ன பெயர் வைப்பது?

```text

```

# ஏன் positive number மட்டும் √2?

ஏனெனில்

```text
x² = 2
```

க்கு இரண்டு answers இருக்கின்றன.

```text
√2
```

மற்றும்

```text
-√2
```

இரண்டு பேருக்கும் ஒரே பெயர் வைத்தால் குழப்பம் வரும்.

அதனால் convention:

* Positive answer → `√2`
* Negative answer → `-√2`

---

# ஒரு ஒப்புமை

ஒரு வீட்டில் இரட்டை சகோதரர்கள் இருக்கிறார்கள்.

* அருண்
* -அருண் ❌ (இப்படி பெயர் வைக்க மாட்டோம்!)

அவர்களுக்கு தனித்தனி பெயர் வேண்டும்.

அதேபோல்

* Positive root → `√2`
* Negative root → `-√2`

# இது நிஜ வாழ்க்கையில் எங்கே வருகிறது?

ஒரு **1 × 1** சதுரத்தை எடுத்துக்கொள்.

```
+------+
|     /|
|   /  |
| /    |
+------+
```

Pythagoras theorem படி,

மூலைவிட்டம்:

```text
d² = 1² + 1²
```

```text
d² = 2
```

அதனால்

```text
d = √2
```

இப்போது அந்த மூலைவிட்டத்தின் நீளம் **√2**.

இது ஒரு கற்பனை அல்ல. உண்மையான அளவு.

## இந்தப் பக்கத்தின் மிகப் பெரிய கருத்து

Serge Lang உனக்கு ஒரு புதிய operation கற்றுக்கொடுக்கவில்லை.

அவர் ஒரு **புதிய எண்ணை அறிமுகப்படுத்துகிறார்.**

அவர் சொல்வது:

> "இனி `x² = 2` என்றால், அதன் positive solution-க்கு `√2` என்று பெயர் வைப்போம்."
