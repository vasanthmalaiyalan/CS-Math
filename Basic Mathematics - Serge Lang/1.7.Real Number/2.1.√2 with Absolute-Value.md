`2 நடுவில் என்ன என்பதை கண்டுபிடிப்பதற்காக √2 என்பதை பயன்படுத்தினோம் ஆனால் இந்த பகுதியில் அனைத்திற்கும் √ இதை நாம் பயன்படுத்துகிறோமா? like x=-3 என்றால் x^2=9 என்பதை இப்போது √9 என்று பயன்படுத்துகிறோமா இப்படித்தான் பயன்படுத்த வேண்டுமா?`

அருமையான கேள்வி. 👏 இந்த இடத்தில்தான் பல மாணவர்கள் குழப்பமடைகிறார்கள்.

பதில்: **ஆம், ஆனால் ஒரு முக்கியமான விதியுடன்.**

---

## முதலில் இதைப் பார்

உனக்குத் தெரியும்:

```text
x = -3
```

அப்போது

```text
x² = (-3)² = 9
```

இது சரிதானே?

---

## இப்போது இருபக்கத்துக்கும் square root எடுக்கலாமா?

ஆம்.

```text
√(x²) = √9
```

இங்கேயே முக்கியமான விஷயம் வருகிறது.

புத்தகம் சொல்வது:

```text
√(x²) = |x|
```

**`x` அல்ல.**

---

### ஏன்?

ஏனெனில்

```text
x = -3
```

என்றால்

இடது பக்கம்:

```text
√((-3)²)
=
√9
=
3
```

ஆனால்

```text
x = -3
```

அதனால்

```text
√(x²)
=
3
≠
-3
```

எனவே

```text
√(x²)=x
```

❌ தவறு.

சரியானது:

```text
√(x²)=|x|
```

---

## இப்போது உன் கேள்விக்கு நேரடி பதில்

நீ கேட்டாய்:

> **"x=-3 என்றால் x²=9. இப்போது √9 என்று பயன்படுத்துகிறோமா?"**

**ஆம்.**

ஏனெனில்

```text
x² = 9
```

என்ற equation-இல் இருந்து மீண்டும் `x`-ஐ பெற வேண்டும்.

அதற்கு square root operation பயன்படுத்துகிறோம்.

```text
√(x²) = √9
```

↓

```text
|x| = 3
```

இப்போது

```text
|x| = 3
```

என்றால்

இரண்டு possibilities:

```text
x = 3
```

அல்லது

```text
x = -3
```

---

## இன்னொரு உதாரணம்

Suppose

```text
x² = 16
```

இப்போது square root எடுப்போம்.

```text
√(x²) = √16
```

↓

```text
|x| = 4
```

இதன் அர்த்தம்

```text
x = 4
```

அல்லது

```text
x = -4
```

## முக்கியமான விதி

Square root (`√`) என்பது **square-ன் எதிர் (inverse) operation**.

உதாரணம்:

```text
5² = 25
```

அதைத் திருப்பிப் பெற:

```text
√25 = 5
```

---

ஆனால்

```text
(-5)² = 25
```

என்றாலும்

```text
√25
```

எப்போதும்

```text
5
```

தான்.

**-5 அல்ல.**

ஏனெனில் **√ என்ற குறியீடு எப்போதும் positive root-ஐ மட்டும் குறிக்கும்**.

---

### இதை ஒரு வரியில் நினைவில் வை

**Square** இரண்டு எண்களை (உதா: `+3`, `-3`) ஒரே எண்ணாக (`9`) மாற்றிவிடும்.

**Square root** அந்த ஒரு எண்ணிலிருந்து **positive root**-ஐ மட்டும் நேரடியாகத் தரும்.

அதனால் equation solve செய்யும்போது நாம்

```text
|x| = √9 = 3
```

என்று எழுதி, அதன் பிறகுதான்

```text
x = ±3
```

என்று முடிவு செய்கிறோம்.

---

**நான் ஒரு கேள்வி கேட்கிறேன்.**

இருபக்கத்துக்கும் square root எடுத்தால்,


# இப்போது முக்கியமான விஷயம்

நீ கேட்டது:

> **"இனிமேல் x² வந்தால் எல்லாம் √ போடணுமா?"**

**பதில்: இல்லை.**

நாம் **√-ஐ போடுவது x-ஐ மீண்டும் கண்டுபிடிக்க வேண்டியபோது மட்டும்.**

### Example 1

```text
x = -3
```

அப்போது

```text
x² = 9
```

இங்கே முடிந்துவிட்டது.

❌ இங்கே √ போட வேண்டிய அவசியமில்லை.

---

### Example 2

ஆனால் கேள்வி

```text
x² = 9
```

**x என்ன?**

என்று கேட்டால்,

அப்போதுதான்

```text
√(x²)=√9
```

என்று square root எடுக்கிறோம்.

---

## நினைவில் வைத்துக்கொள்ள ஒரு விதி

**Square** என்பது:

```text
x → x²
```

உதாரணம்:

```text
3 → 9
```

---

**Square Root** என்பது:

```text
x² → x-ஐ கண்டுபிடிக்க
```

உதாரணம்:

```text
49 → 7
```

அதனால் **√ என்பது "பின்னோக்கி செல்லும் (reverse)" operation**.

---

### ஒரு சிறிய ஒப்புமை

நீ ஒரு முட்டையை உடைத்து ஆம்லெட் செய்தாய்.

* முட்டை → ஆம்லெட் (Square)
* ஆம்லெட்டைப் பார்த்து "இது எந்த முட்டையிலிருந்து வந்தது?" என்று கண்டுபிடிப்பது (Square Root)

அதுபோல,

* `x → x²`
* `x² → √` பயன்படுத்தி மீண்டும் `x`-ஐத் தேடுகிறோம்.

---

```text
x² = 100
```
