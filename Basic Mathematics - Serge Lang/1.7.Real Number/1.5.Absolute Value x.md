அருமை. இந்தப் பக்கம் (**Page 66**) தான் **Absolute Value (`|x|`)** என்ற கருத்தை முதன்முறையாக அறிமுகப்படுத்துகிறது.

இந்தப் பக்கத்தில் **5 பெரிய கருத்துகள்** உள்ளன. ஒவ்வொன்றாகப் பார்ப்போம்.

---

# Concept 1: ஏன்

என்று எழுதுகிறார்?

இதை நீ ஏற்கனவே புரிந்துவிட்டாய்.

உதாரணம்

```text
x² = 9
```

என்றால்

```text
x = 3
```

ஏனெனில்

```text
3² = 9
```

மேலும்

```text
x = -3
```

ஏனெனில்

```text
(-3)² = 9
```

அதனால்

```text
x = ±3
```

---

## General Rule

எந்த positive number `a` இருந்தாலும்

```text
x² = a
```

என்றால்

Answer

```text
x = ±√a
```

# Concept 2

புத்தகம் சொல்கிறது

```text
If

x²=y²

then

x=y

or

x=-y
```

ஏன்?

Example

```text
5²=25
```

மேலும்

```text
(-5)²=25
```

அதனால்

```text
5²=(-5)²
```

ஆனால்

```text
5≠-5
```

இதிலிருந்து

ஒரு முக்கியமான விஷயம் தெரிகிறது.

---

**Square equal என்றால்**

எண்கள் equal ஆகவேண்டும் என்று சொல்ல முடியாது.

அவை

* equal

அல்லது

* opposite

ஆக இருக்கலாம்.

---

Example

```text
7²=(-7)²
```

ஆனால்

```text
7≠-7
```

# Concept 3

இப்போதுதான் மிகவும் முக்கியமான கருத்து வருகிறது.


Positive.

## இந்தப் பக்கத்தின் ஒரு வரியில் சுருக்கம்

இந்த Page-ன் முக்கியமான theorem:

```text
√(x²)=|x|
```
