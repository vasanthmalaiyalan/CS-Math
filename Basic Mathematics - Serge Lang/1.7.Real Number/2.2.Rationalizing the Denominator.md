# இந்த Page-ன் முக்கிய கருத்துகள்

1. `1/√2 = √2/2` — numerator, denominator இரண்டையும் `√2`-ஆல் பெருக்குவதால்.
2. **Rationalize the denominator** என்றால் denominator-ல் இருக்கும் square root-ஐ நீக்குவது.
3. அதை செய்ய **conjugate**-ஐப் பயன்படுத்துகிறோம்.
4. ஏன் conjugate? ஏனெனில் `(a+b)(a-b)=a²-b²`, square root மறைந்து rational number கிடைக்கும்.
5. இந்த technique பின்னர் Algebra, Geometry, Calculus ஆகியவற்றில் அடிக்கடி பயன்படும்.

## இப்போது உனக்கான ஒரு சிறிய பயிற்சி

நான் கேட்கிறேன்:

```text
      1
------------
   3+√2
```

இதன் **conjugate** என்ன?

அதை மட்டும் சொல்லு. முழு answer வேண்டாம்.

--------------

`நீ சொன்னதில் இது மட்டும் புரியவில்லை `6-3√5
-----------
-1` -1 வகுத்தால் எப்படி இந்த பதில் கிடைக்கும் `-6+3√5`?`

அருமை. இந்த இடத்தில்தான் உனக்கு குழப்பம் வந்திருக்கிறது. **இங்கே √ (root) சம்பந்தமே இல்லை.** இது **negative number-ஆல் division** மட்டும்.

### நினைவில் வைத்துக்கொள்ள வேண்டிய ஒரு சிறிய விதி

**ஒரு முழு expression-ஐ `-1`-ஆல் divide செய்தால், expression-ல் உள்ள ஒவ்வொரு term-ன் sign-மும் மாறும்.**

உதாரணம்:

```text
(a+b)/(-1)
=
-a-b
```

```text
(a-b)/(-1)
=
-a+b
```

அதனால்தான்

```text
(6-3√5)/(-1)
=
-6+3√5
```

ஆகிறது. இது **division by -1** மட்டுமே; இதில் square root-க்கு எந்தப் புதிய விதியும் இல்லை.
