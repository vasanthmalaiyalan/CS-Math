## இந்த Exercise-ஐ ஒரே Rule-ல் நினைவில் வை

ஒவ்வொரு கேள்வியிலும் செய்ய வேண்டியது இதே:

1. **Numerator-ன் conjugate கண்டுபிடி.**
2. **மேலும் கீழும் அதே conjugate-ஆல் பெருக்கு.**
3. **`(a+b)(a-b)=a²-b²` பயன்படுத்தி square root-ஐ நீக்கு.**
4. **`h` அல்லது common factor இருந்தால் cancel செய்.**
5. **Final simplified answer எழுது.**

இந்த ஆறு கேள்விகளும் உண்மையில் **ஒரே technique**-ன் வெவ்வேறு வடிவங்கள் மட்டுமே.

அருமை! இதுதான் முக்கியமான இடம். நீ **Difference of Squares** formula-வை மனப்பாடம் செய்யாமல், அது எப்படி வேலை செய்கிறது என்று புரிந்துகொள்ள விரும்புகிறாய். அதுதான் சரியான அணுகுமுறை.

முதலில் இந்தப் பகுதியைப் பார்:

[
(\sqrt{2x+3}+1)(\sqrt{2x+3}-1)
]

இங்கே

```text
A = √(2x+3)

B = 1
```

ஆக உள்ளது.

அதனால் இது

```text
(A+B)(A-B)
```

என்ற வடிவத்தில் இருக்கிறது.

## Step 1

Formula-வில்

```text
a = √(2x+3)

b = 1
```

என்று substitute செய்கிறோம்.

அப்போது

```text
(a+b)(a-b)
```

↓

```text
(√(2x+3)+1)(√(2x+3)-1)
```

↓

```text
(√(2x+3))² - (1)²
```

---

## Step 2

இப்போது ஒவ்வொன்றாக square செய்கிறோம்.

```text

```

```text
(√A)² = A
```

அதனால்

```text

```

## Verification (FOIL method)

Formula மீது நம்பிக்கை வைக்காமல், முழுவதும் விரித்து எழுதுவோம்.

```text
(√(2x+3)+1)(√(2x+3)-1)
```

முதல் term:

```text
√(2x+3) × √(2x+3)
=
2x+3
```

Outer:

```text
√(2x+3) × (-1)
=
-√(2x+3)
```

Inner:

```text
1 × √(2x+3)
=
+√(2x+3)
```

Last:

```text
1 × (-1)
=
-1
```

இப்போது அனைத்தையும் சேர்க்கிறோம்.

```text
2x+3
-√(2x+3)
+√(2x+3)
-1
```

நடுவிலுள்ள இரண்டு terms cancel ஆகிவிடும்.

```text
-√(2x+3)
+
√(2x+3)
=
0
```

மீதம்

```text
2x+3-1
=
2x+2
```

அதே answer வந்தது.

### நினைவில் வைக்க வேண்டிய முக்கிய Rule

எப்போதெல்லாம்

```text
(√A+1)(√A-1)
```

போன்ற expression பார்த்தாலும், அது எப்போதும்

```text
A-1
```

ஆக மாறும்.

அதேபோல்

```text
(√A+√B)(√A-√B)
```

என்றால்

```text
A-B
```

ஆக மாறும்.

இதுதான் conjugate-ஐ பயன்படுத்துவதற்கான முழு காரணம். Square root terms நடுவில் cancel ஆகி, square root இல்லாத expression கிடைக்கிறது.
