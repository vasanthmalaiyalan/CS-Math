``x² = 2`என்றால் x = 1.414213562373095... இதுபோல ஆன்சர் வருவதால் நாம் x= √2 என்று பயன்படுத்தினோம் இதுவே x^2=9 என்று இருந்தால் √ root பயன்படுத்த தேவையில்லை தானே x= 3 என்று மட்டும் போடலாமே? `

இது மிகச் சிறந்த கேள்வி. உண்மையில் இதைப் புரிந்துகொண்டால், **√ (square root)** பற்றிய 90% குழப்பம் தீர்ந்துவிடும்.

பதில்:

**ஆம், `x² = 9` என்றால் நேரடியாக `x = 3` என்று மட்டும் எழுதக்கூடாது.**

ஏன் என்று பார்ப்போம்.

---

## Case 1: `x² = 9`

கேள்வி:

> எந்த எண்ணை square செய்தால் 9 வரும்?

நாம் அறிவோம்:

```text
3² = 9
```

மேலும்

```text
(-3)² = 9
```

அதனால் answer:

```text
x = ±3
```

இங்கே **√9 = 3** என்று எழுதலாம்.

ஆனால் கவனிக்க வேண்டும்:

```text
√9 = 3
```

இது **positive root மட்டும்.**

அதனால் equation solve செய்யும்போது:

```text
√(x²) = √9
```

↓

```text
|x| = 3
```

↓

```text
x = ±3
```

---

## Case 2: `x² = 2`

இங்கே பிரச்சினை.

நமக்கு 2-ன் square root-ஆக ஒரு முழு எண் தெரியாது.

அதனால் ஒரு புதிய பெயர் வைத்தோம்:

```text
√2
```

அதனால் answer:

```text
x = ±√2
```

---

## முக்கியமான உண்மை

**√2 என்பது "2-க்காக மட்டும்" உருவாக்கப்பட்ட symbol அல்ல.**

**√ (square root)** என்பது **எல்லா positive எண்களுக்கும்** பயன்படும் operation.

உதாரணம்:

```text
√4 = 2
```

```text
√9 = 3
```

```text
√16 = 4
```

```text
√25 = 5
```

இங்கே answers நமக்குத் தெரியும்.

அதனால் decimal எழுத வேண்டிய அவசியம் இல்லை.

---

ஆனால்

```text
√2
```

```text
√3
```

```text
√5
```

```text
√7
```

இவற்றின் மதிப்பு முழு எண்களாக வராது.

அதனால் **√ வடிவத்திலேயே** விடுகிறோம்.

## ஒரு முக்கியமான கருத்து

நீ இப்படிக் கேட்கலாம்:

> **"அப்படியானால் √9 என்று ஏன் எழுத வேண்டும்? நேரடியாக 3 என்று எழுதலாமே?"**

ஆம், **எழுதலாம்.**

உதாரணமாக,

```text
√9 = 3
```

இங்கே `√9`-ஐ உடனே `3` என்று மாற்றலாம்.

ஆனால் `√2`-ஐ அப்படி மாற்ற முடியாது.

ஏனெனில் அதன் exact value:

```text
1.414213562...
```

முடிவில்லாத decimal.

அதனால் **exact answer** ஆக

```text
√2
```

என்றே எழுதுகிறோம்.

### இதை ஒரு வரியில் நினைவில் வை

**√ என்பது "2-க்காக மட்டும்" உருவாக்கப்பட்ட symbol அல்ல.**

**√ என்பது square-க்கு எதிரான (inverse) operation.**

* `√9 = 3` (மதிப்பு எளிதாகத் தெரியும்)
* `√16 = 4` (மதிப்பு எளிதாகத் தெரியும்)
* `√2` (மதிப்பு முழு எண் அல்ல; அதனால் symbol-ஆகவே விடுகிறோம்)

அதனால்தான் `√9`-யும் `√2`-யும் **ஒரே operation**, வித்தியாசம் என்னவென்றால் ஒன்றின் மதிப்பு உடனே தெரியும், மற்றொன்றின் மதிப்பு முடிவில்லாத decimal ஆக இருக்கும்.
