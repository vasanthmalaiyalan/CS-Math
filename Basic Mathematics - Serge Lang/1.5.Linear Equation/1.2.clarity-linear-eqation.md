```text
(30x+30y)
=
400-240
```

இடது பக்கம்:

```text
60x-30x
=
30x
```

```text
30y-30y
=
0
```

ஆக

```text
30x=160
```

## Step 5: x கண்டுபிடி

```text
x=160/30
```

Simplify:

```text
x=16/3
```

## Step 6: y கண்டுபிடி

முதல் equation:

```text
x+y=8
```

x substitute செய்:

```text
16/3+y=8
```

8-ஐ fraction-ஆக எழுது:

```text
8=24/3
```

ஆக

```text
16/3+y=24/3
```

இருபக்கத்திலும்

```text
-16/3
```

சேர்:

```text
y
=
24/3-16/3
```

```text
=
8/3
```

## Answer

```text
x = 16/3
```

```text
y = 8/3
```

### இந்த method-ன் முக்கியமான கருத்து

நாம் **guess செய்யவில்லை.**

நாம் செய்தது:

1. இரண்டு equations எழுதினோம்.
2. ஒரு variable-ன் coefficient-ஐ சமமாக்கினோம்.
3. Subtract செய்து ஒரு variable-ஐ eliminate செய்தோம்.
4. மீதமுள்ள variable-ஐ கண்டுபிடித்தோம்.
5. அதை மீண்டும் substitute செய்து மற்ற variable-ஐ கண்டுபிடித்தோம்.

இதுதான் **Elimination Method**.

இதுவே Serge Lang இந்த chapter-ல் கற்றுக்கொடுக்கும் முக்கியமான technique.
