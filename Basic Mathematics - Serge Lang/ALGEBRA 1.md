## இந்த Chapter-ன் நோக்கம் என்ன?

இந்த Chapter-ல் ஆசிரியர் எண்களுடன் (Numbers) நாம் செய்யும் அடிப்படை செயல்களை முறையாக கற்றுக்கொடுக்கப் போகிறார்.

உதாரணம்:

* கூட்டல் (+)
* பெருக்கல் (×)
* வகுத்தல் (÷)
* சமமின்மை (<, >)
* நேர்மறை / எதிர்மறை எண்கள்
* Square Root (√)
* n-th Root

போன்றவை.

## Algebra விதிகள் ஏன் முக்கியம்?

சில விதிகள் கணிதத்தின் எல்லா இடங்களிலும் மீண்டும் மீண்டும் வரும்.

உதாரணம்:

### Commutative Law

2 + 3 = 3 + 2

5 × 7 = 7 × 5

வரிசை மாறினாலும் பதில் மாறாது.

---

### Associative Law

(2 + 3) + 4 = 2 + (3 + 4)

(2 × 3) × 4 = 2 × (3 × 4)

Grouping மாறினாலும் பதில் மாறாது.

---

இந்த விதிகள் வெறும் Numbers-க்கு மட்டும் அல்ல.

பிறகு:

* Complex Numbers
* Functions
* Mappings
* Vectors

போன்றவற்றிலும் பயன்படுத்தப்படும். 

---

## Algebra மற்றும் Geometry எப்படி இணைகின்றன?

Geometry படிக்கும்போதும் Algebra தேவை.

உதாரணம்:

ஒரு Rectangle-ன் Area:

Area = Length × Width

ஒரு Triangle-ன் Area:

Area = ½ × Base × Height

இவை எல்லாம் Numbers-ஐ பயன்படுத்துகின்றன.

அதனால் Algebra மற்றும் Geometry ஒன்றோடொன்று கலந்து இருக்கும்.

## ஏன் புதிய Number Systems உருவாக்கப்பட்டன?

சில Equations பழைய Number System-ல் Solve ஆகாது.

---

### Integer-ல்

```
2x = 3
```

Solve செய்ய முடியாது.

ஏனெனில்

```
x = 3/2
```

Integer இல்லை.

---

### Rational Number-ல்

```
x² = 2
```

Solve செய்ய முடியாது.

ஏனெனில்

```
x = √2
```

Rational Number இல்லை.

---

### Real Number-ல்

```
x² = -1
```

Solve செய்ய முடியாது.

அதற்காக Complex Numbers உருவாக்கப்பட்டன.

```
i = √(-1)
```

## ஆசிரியர் சொல்லும் பெரிய Mathematical Idea

கணிதத்தில் அடிக்கடி இதுதான் நடக்கும்:

1. ஒரு System இருக்கும்.
2. அதில் சில Problems Solve ஆகாது.
3. அதைவிட பெரிய System உருவாக்கப்படும்.
4. அந்த Problem Solve ஆகும்.

உதாரணம்:

```
Integers
   ↓
Rational Numbers
   ↓
Real Numbers
   ↓
Complex Numbers
```

இது கணிதத்தின் மிகவும் முக்கியமான சிந்தனை முறைகளில் ஒன்று.
