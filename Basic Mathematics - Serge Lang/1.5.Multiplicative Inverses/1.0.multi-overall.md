# 1. Division என்பது உண்மையில் Multiplication by Inverse

Lang சொல்கிறார்:

```text
a/b
```

என்பது உண்மையில்

அதாவது

```text
a × (1/b)
```

உதாரணம்:

```text
3/4
=
3 × (4⁻¹)
=
3 × 1/4
```

இதனால் division என்பது அடிப்படையில் புதிய operation அல்ல. Multiplication மட்டுமே.

---

# 2. Rational Cross Multiplication

Lang நிரூபிக்கிறார்:

ஏன்?

ஏனெனில்

```text
a/b = c/d
```

என்பது

```text
ab⁻¹ = cd⁻¹
```

இருபக்கத்தையும் `bd`-ஆல் பெருக்கு:

```text
abd·b⁻¹ = cbd·d⁻¹
```

ஆனால்

```text
bb⁻¹=1
dd⁻¹=1
```

எனவே

```text
ad=bc
```

---

# 3. Multiplication Cancellation

Lang சொல்கிறார்:

If

```text
ab=ac
```

and

```text
a≠0
```

then

```text
b=c
```

ஏன்?

இருபக்கத்தையும் `a⁻¹`-ஆல் பெருக்கு:

```text
a⁻¹ab=a⁻¹ac
```

Associativity:

```text
(a⁻¹a)b=(a⁻¹a)c
```

```text
1b=1c
```

அதனால்

```text
b=c
```

இதேதான் multiplication version of cancellation.

---

# 4. Rational Fraction Addition

Lang derive செய்கிறார்:

ஏன்?

Common denominator வேண்டும்.

```text
a/b = ad/bd
c/d = bc/bd
```

அதனால்

```text
(ad+bc)/bd
```

வருகிறது.

---

# 5. Example

Lang காட்டுகிறார்:

முதலில் common denominator:

```text
(x-y)(x+y)
```

Numerator:

```text
(x+y)+(x-y)
```

இப்போது:

```text
x+y+x-y
=
2x
```

Denominator:

```text
(x-y)(x+y)
=
x²-y²
```

ஆக

```text
2x/(x²-y²)
```

---

# 6. Solving Rational Equations

Lang:

```text
(3x+1)/(2x-5)=4
```

Cross multiply:

```text
3x+1
=
4(2x-5)
```

Distribute:

```text
3x+1=8x-20
```

Additive inverse:

```text
21=5x
```

Multiplicative inverse:

```text
x=21/5
```

# 7. Physics

Lang:

```text
d=st
```

Distance = speed × time

அதனால்

```text
s=d/t
```

ஏன்?

t-ன் multiplicative inverse:

```text
1/t
```

இருபக்கத்தையும் `1/t`-ஆல் பெருக்கு:

```text
d·1/t
=
st·1/t
```

```text
d/t=s
```

# 8. Town Problem

Speed in town = 30 mph

Freeway = 60 mph

Total time = 8 hr

Let

```text
x = town time
```

Freeway time:

```text
8-x
```

Distance:

```text
30x
```

Freeway distance:

```text
60(8-x)
```

Total distance:

```text
30x+60(8-x)=400
```

Expand:

```text
30x+480-60x=400
```

```text
80=30x
```

```text
x=8/3
```

---

# 9. Antifreeze Problem

Initially:

8 qt

40% antifreeze

Amount drained = x

Remaining liquid:

```text
8-x
```

40% antifreeze remains:

```text
(40/100)(8-x)
```

Add x qt pure antifreeze:

```text
x+(40/100)(8-x)
```

Final mixture should be 90% of 8 qt:

```text
(90/100)8
```

Equation:

```text
x+(40/100)(8-x)
=
(90/100)8
```

Expand:

```text
x+3.2-0.4x=7.2
```

```text
0.6x=4
```

```text
x=20/3
```

> * cancellation
> * algebra equations
> * speed formulas
> * mixture problems
>
