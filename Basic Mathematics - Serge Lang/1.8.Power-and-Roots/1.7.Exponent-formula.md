`2^1x2^(1/2) இப்படி இருக்கிறது எப்படி 2^(1+1/2) ஆனது exponent fromla இருப்பது இது `2^1x2^(1/2) `இப்படியாகத்தா `4^(1+1/2)`?`

நீ கேட்ட கேள்வி **மிகச் சரியானது**. இந்த இடத்தில்தான் பலர் தவறு செய்கிறார்கள்.

நீ கேட்டது:

> ஏன்

```text
2¹ × 2^(1/2)
```

இது

```text
2^(1+1/2)
```

ஆகிறது?

ஏன்

```text
4^(1+1/2)
```

ஆகவில்லை?

---

## காரணம்

Exponent Rule-ல் **Base (அடிப்படை எண்) ஒன்றாக இருக்க வேண்டும்.**

Rule:

கவனிக்க:

இங்கே

```text
a^x
```

மற்றும்

```text
a^y
```

**இரண்டிலும் base = a.**

---

### நம்முடைய உதாரணம்

```text
2¹ × 2^(1/2)
```

முதல் term:

```text
2¹
```

Base = **2**

---

இரண்டாவது term:

```text
2^(1/2)
```

Base = **2**

---

இரண்டிலும் base **ஒரே 2**.

அதனால் exponent-களை மட்டும் கூட்டலாம்.

```text
2^(1+1/2)
```

---

## ஏன் 4 வரக்கூடாது?

நீ ஒருவேளை இப்படி நினைத்திருக்கலாம்:

```text
2¹ = 2
```

மேலும்

```text
2 × 2
=
4
```

அதனால்

```text
4^(1+1/2)
```

என்று.

ஆனால் இது exponent rule அல்ல.

ஏனெனில் exponent rule சொல்வது:

> **"Base-ஐ மாற்றக்கூடாது."**

---

## ஒரு எளிய உதாரணம்

```text
3² × 3⁴
```

Base:

```text
3
```

மற்றும்

```text
3
```

அதனால்:

```text
3^(2+4)
=
3⁶
```

**இது**

```text
9⁶
```

ஆகாது.

ஏனெனில் 9 என்பது base இல்லை.

=


**இது**

```text

```

ஆகாது.

## அதனால்

```text
2¹ × 2^(1/2)
```

இங்கே:

* Base = 2
* Base = 2

அதனால்:

```text
2^(1+1/2)
```

---

**`4^(1+1/2)` என்று எழுத வேண்டுமென்றால்**, முதலில் base-ஐ உண்மையாகவே 4-ஆக மாற்ற வேண்டும்.

உதாரணமாக:

```text
4 = 2²
```

என்பதைப் பயன்படுத்தி மாற்றலாம். ஆனால் அது வேறு ஒரு கணக்குப் படி; exponent rule-ஐ நேரடியாகப் பயன்படுத்தும் போது **base எப்போதும் அதேபடியே இருக்கும்**.

---

### ஒரு முக்கியமான விதி

எப்போதும் இந்த இரண்டு கேள்விகளை முதலில் கேள்:

1. **Base ஒன்றா?**

   * ஆம் → exponent-களை கூட்டலாம்.
   * இல்லை → கூட்ட முடியாது.

2. **Exponent ஒன்றா?**

   * ஆம் → அப்போது `(ab)^x = a^x b^x` போன்ற வேறு விதிகளைப் பயன்படுத்தலாம்.
   * இல்லை → இந்த விதி பொருந்தாது.

இதுதான் exponent rules-ல் குழப்பம் வராமல் இருக்க வேண்டிய அடிப்படை சிந்தனை.
