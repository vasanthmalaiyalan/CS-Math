இந்த Exercise முழுவதும் **fractional exponent** (`1/2`, `3/2`, `2/3`, `1/4`...) சரியாகப் புரிந்ததா என்பதைச் சோதிக்கிறது.

நாம் ஒன்றொன்றாக Logic-உடன் பார்ப்போம்.

---

# 2. What integer is (81^{1/4}) equal to?

முதலில்

```text
81 = 3⁴
```

இப்போது

```text
81^(1/4)
=
(3⁴)^(1/4)
```

Rule:

```text
(a^m)^n = a^(mn)
```

அதனால்

```text
=
3^(4×1/4)
=
3¹
=
3
```

✅ **Answer = 3**

---

# 3. What integer is ((√2)^6) equal to?

நமக்கு தெரியும்

```text
√2
=
2^(1/2)
```

அதனால்

```text
(√2)^6
=
(2^(1/2))^6
```

Rule:

```text
(a^x)^y
=
a^(xy)
```

```text
=
2^(6/2)
=
2³
=
8
```

✅ **Answer = 8**

# 4. Is ((√2)^5) an integer?

```text
(√2)^5
=
(2^(1/2))^5
=
2^(5/2)
```

அல்லது

```text
=
(√2)^4×√2
=
4√2
```

இப்போது

```text
√2
```

irrational.

அதனால்

```text
4√2
```

integer இல்லை.

❌ **Answer = No**

## இந்த Exercise-ன் முக்கிய நோக்கம்

இந்த Chapter முழுவதும் ஒரு **Golden Rule**-ஐப் பயன்படுத்தச் செய்கிறது:

```text
a^(m/n)
```

என்றால் எப்போதும் இதுபோல் யோசி:

1. **முதலில் n-th root எடு.**
2. **பிறகு m-th power எடு.**

அல்லது,

1. **முதலில் m-th power எடு.**
2. **பிறகு n-th root எடு.**

இரண்டு வழிகளும் ஒரே answer-ஐத் தரும். இதுதான் fractional exponent-ன் மிக முக்கியமான property.
