``√2
=
2^(1/2)` இது செயற்கையாக  நாம் போடப்பட்டோமா இல்லை லாஜிக் உள்ளதா?`

மிகச் சிறந்த கேள்வி. இதுதான் கணிதத்தின் முக்கியமான திருப்புமுனை.

உன் கேள்வி:

> **`√2 = 2^(1/2)` இது சும்மா mathematicians வைத்த notation-ஆ? அல்லது இதற்குப் பின்னால் logic இருக்கிறதா?**

பதில்:

> **இது வெறும் notation அல்ல. இதற்குப் பின்னால் மிகவும் வலுவான logic இருக்கிறது.**

---

## Step 1: முதலில் square root-ன் definition

Square root என்றால் என்ன?

```text
√2
```

என்பது

> **"எந்த positive எண்ணை square செய்தால் 2 வரும்?"**

அந்த எண்ணே

```text
√2
```

அதாவது

```text
(√2)² = 2
```

இதுதான் definition.

---

## Step 2: Exponent என்ன சொல்கிறது?

நாம் ஏற்கனவே தெரிந்தது:

```text
(a^m)^n = a^(mn)
```

இது Serge Lang புத்தகத்திலும் இருக்கிறது.

---

## Step 3: இப்போது ஒரு புதிய notation உருவாக்க வேண்டும்

Mathematicians யோசித்தார்கள்.

ஒரு எண் வேண்டும்.

அதை

```text
a^x
```

என்று எழுத வேண்டும்.

அதற்கு

```text
(a^x)² = a
```

வர வேண்டும்.

அப்போது

```text
(a^x)²
=
a^(2x)
```

இது exponent rule.

இப்போது

```text
a^(2x)=a¹
```

வர வேண்டும்.

அதனால்

```text
2x=1
```

அதனால்

```text
x=1/2
```

கிடைக்கிறது.

அதனால்

```text
a^(1/2)
```

என்ற notation-ஐ வைத்தால்

```text
(a^(1/2))²
=
a
```

இது square root-ன் definition-க்கு **அப்படியே பொருந்துகிறது.**

அதனால்

```text
√a
=
a^(1/2)
```

என்று எழுதுகிறார்கள்.

---

## Verify பண்ணிப் பார்ப்போம்

எடுத்துக்கொள்

```text
9^(1/2)
```

Square செய்.

```text
(9^(1/2))²
=
9^((1/2)×2)
=
9¹
=
9
```

அப்படியானால்

```text
9^(1/2)
```

என்பது

9-க்கு square root ஆக இருக்க வேண்டும்.

அது

```text
3
```

தான்.

## அதே Logic-ல் Cube Root

Cube root என்றால்

```text
∛8
```

Definition:

```text
(∛8)³=8
```

ஒரு exponent

```text
a^x
```

என்று எழுத வேண்டுமென்றால்

```text
(a^x)³=a
```

Exponent rule:

```text
a^(3x)=a¹
```

அதனால்

```text
3x=1
```

```text
x=1/3
```

அதனால்

```text
∛a=a^(1/3)
```

## இதுதான் General Formula

n-th root என்றால்

```text
(a^(1/n))^n
=
a
```

ஏனெனில்

```text
a^((1/n)×n)
=
a¹
=
a
```

அதனால்

```text
ⁿ√a
=
a^(1/n)
```

# Serge Lang என்ன செய்தார்?

இது மிகவும் முக்கியம்.

அவர்

> "நாம் `√a = a^(1/2)` என்று **சும்மா** அறிவிக்கிறோம்"

என்று சொல்லவில்லை.

அவர் முதலில்

```text
rⁿ=a
```

என்ற n-th root-ன் definition-ஐ கொடுக்கிறார்.

பிறகு

```text
r=a^(1/n)
```

என்று **notation** கொடுக்கிறார்.

ஏன் அந்த notation?

ஏனெனில் அது exponent rules-ஐ உடைக்காமல், தொடர்ச்சியாக (consistent-ஆக) வைத்திருக்கிறது.

---

## முடிவு

அதனால்

```text
√2 = 2^(1/2)
```

என்பது

❌ "செயற்கையாக வைத்த notation" அல்ல.

✅ "Exponent rules (`(a^x)^y = a^(xy)`) மற்றும் root-ன் definition (`r²=a`) இரண்டும் ஒரே நேரத்தில் உண்மையாக இருக்க வேண்டும்" என்ற **கணித லாஜிக்கால் கட்டாயமாக வந்த notation**.

**அதனால்தான்** Serge Lang போன்ற புத்தகங்களில் புதிய notation-கள் பெரும்பாலும் "அழகாகத் தோன்றுகிறது" என்பதற்காக வருவதில்லை; ஏற்கனவே உள்ள விதிகளுடன் முரண்படாமல் இருக்க வேண்டிய அவசியத்திலிருந்து உருவாகின்றன.

----------------------

``√2 = 2^(1/2)`அதாவது √ இதற்கு பதிலாக 1/2 என்று போடுகிறோம் 1/2 square செய்தால் மீண்டும் இரண்டு வருகிறது`


நீ சொன்னதில் **ஒரு சிறிய தவறு** இருக்கிறது. அதைத்தான் சரி செய்தால் முழு concept புரிந்துவிடும்.

நீ சொன்னது:

> **"`√`-க்கு பதிலாக `1/2` போடுகிறோம். `1/2` square செய்தால் மீண்டும் 2 வருகிறது."**

❌ இது சரியில்லை.

**`1/2`-ஐ square செய்வதில்லை.**

**`2^(1/2)`-ஐ square செய்கிறோம்.**

அதுதான் மிக முக்கியமான வித்தியாசம்.

---

பார்.

நாம் ஒரு புதிய notation உருவாக்க வேண்டும்.

அதன் பெயர்:

```text
2^(1/2)
```

இது எப்படி இருக்க வேண்டும்?

இதை square செய்தால்

```text
(2^(1/2))²
```

Exponent rule பயன்படுத்து:

அதனால்

```text
(2^(1/2))²
=
2^((1/2)×2)
=
2¹
=
2
```

இதுதான் காரணம்.

---

## இதை √ உடன் ஒப்பிடு

Square root-ன் definition என்ன?

```text
(√2)² = 2
```

இப்போது

```text
(2^(1/2))² = 2
```

இதுவும் அதே property-ஐக் கொண்டுள்ளது.

அதனால் mathematicians சொல்கிறார்கள்:

```text
√2 = 2^(1/2)
```

---

## இன்னும் எளிமையாக

`√` என்ற குறியீட்டை அவர்கள் **`1/2`-ஆக மாற்றவில்லை**.

அவர்கள் சொல்வது:

> "**Square root போல நடக்கும் exponent எது?**"

அதை கண்டுபிடிப்போம்.

ஒரு exponent-ஐ `x` என்று வைத்துக்கொள்.

```text
2^x
```

இது square root ஆக இருக்க வேண்டுமென்றால்

```text
(2^x)² = 2
```

வர வேண்டும்.

Exponent rule:

```text
2^(2x)=2¹
```

அதனால்

```text
2x=1
```

```text
x=1/2
```

அதனால்

```text
2^x
=
2^(1/2)
```

இதுதான் square root.

## அதாவது...

**அவர்கள் முதலில் `√2`-ஐப் பார்த்து `2^(1/2)` என்று எழுதவில்லை.**

அவர்கள் இப்படி யோசித்தார்கள்:

> "எந்த exponent வைத்தால் square செய்தவுடன் மீண்டும் original number வரும்?"

கணக்குப் பார்த்தால்

```text
x=1/2
```

என்று வருகிறது.

அதனால்

```text
√2 = 2^(1/2)
```

என்பது **கண்டுபிடிக்கப்பட்டது**, சும்மா **வைக்கப்பட்டது அல்ல**.

இதுதான் அதன் ஆழமான கணித லாஜிக்.
