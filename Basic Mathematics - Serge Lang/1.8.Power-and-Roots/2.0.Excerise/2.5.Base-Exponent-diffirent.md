நல்ல கணிதத்தில்:

* ✅ "இந்த rule ஏன் வேலை செய்கிறது?" என்பதை definition-லிருந்து proof செய்வார்கள்.

**உண்மையில் Serge Lang-ன் புத்தகத்தின் நோக்கமே இதுதான்** — ஒவ்வொரு rule-யும் எந்த definition-லிருந்து வருகிறது என்பதைப் புரியவைத்து, பிறகு தான் அதை வேகமாக பயன்படுத்தும் reflex-ஐ உருவாக்குவது.

# இதுதான் Logic

| Situation             | ஏன் Rule வேலை செய்கிறது?                                                       |
| --------------------- | ------------------------------------------------------------------------------ |
| Base ஒன்றே            | ஒரே பொருளை repeated multiplication செய்கிறோம். அதனால் repetitions சேரும்.      |
| Base ஒன்றே (Division) | Common factors cancel ஆகின்றன.                                                 |
| Exponent ஒன்றே        | ஒவ்வொரு repetition-லேயும் pair செய்ய முடிகிறது.                                |
| இரண்டும் வேறு         | Pair செய்யவும் முடியாது; cancel செய்யவும் முடியாது. அதனால் பொதுவான rule இல்லை. |

## இதுதான் Serge Lang-ன் அழகு

இந்த Rules-ஐ அவர் வானத்திலிருந்து கொண்டு வரவில்லை.

அவை எல்லாம் இரண்டு அடிப்படை properties-லிருந்து வந்தவை:

1. **Repeated multiplication** (`aⁿ`-ன் definition)
2. **Associativity மற்றும் Commutativity**

இந்த இரண்டு இருந்தால், மேலே உள்ள exponent rules அனைத்தும் இயல்பாக வெளிவருகின்றன. அதனால்தான் Serge Lang ஆரம்ப chapter-களில் Associativity, Commutativity ஆகியவற்றை மீண்டும் மீண்டும் வலியுறுத்துகிறார். அவைதான் இந்த proofs-ன் அடித்தளம்.
