``Exponent Rule)

இங்கே denominator-ல் இருப்பது

2⁻³

அதாவது base ஒன்றே (2).

அதனால் இந்த rule-ஐ நேரடியாகப் பயன்படுத்தலாம்.

அதனால்

2⁴
────
2⁻³

=

2^(4-(-3))` incase base is different?`


சரியான கேள்வி! இதுதான் exponent rules-ன் எல்லை (limit).

பதில்:

> **Base வேறாக இருந்தால், இந்த rule-ஐ பயன்படுத்த முடியாது.**

---

## Example 1

```text
2⁴
───
3²
```

இங்கே

* மேலே base = 2
* கீழே base = 3

வேறு வேறு base.

அதனால்

```text
2^(4-2)
```

❌ **என்று எழுத முடியாது.**

---

## ஏன் முடியாது?

நாம் rule-ஐப் பார்ப்போம்.

கவனித்தாயா?

மேலேயும்

```text
a^x
```

கீழேயும்

```text
a^y
```

**இரண்டிலும் `a` ஒரே base.**

அது

```text
2⁴
───
3²
```

இல்லை.

---

## Verify பண்ணிப் பார்ப்போம்

தவறாக

```text
2⁴
───
3²
=
2²
```

என்று வைத்தால்

இடது பக்கம்

```text
16/9
≈1.777...
```

வலது பக்கம்

```text
4
```

சமமா?

இல்லை.

அதனால் rule வேலை செய்யாது.

# Example 5

```text
2⁴
───
3⁻⁴
```

இங்கே

* Base வேறு
* Exponent ஒன்று (-4, 4 என்பது sign வேறுபாடு இருந்தாலும் magnitude ஒன்றல்ல; உண்மையில் numerator exponent = 4, denominator exponent = -4)

முதலில் negative exponent-ஐ மாற்றலாம்.

```text
3⁻⁴
=
1/3⁴
```

அதனால்

```text
2⁴
────
1/3⁴
```

=

```text
2⁴×3⁴
```

இப்போது exponent இரண்டிலும்

```text
4
```

ஒன்றே.

அதனால்

```text
(2×3)⁴
=
6⁴
```

## இதை நினைவில் வைத்துக்கொள்ள ஒரு Trick

ஒவ்வொரு முறையும் இரண்டு கேள்விகள் மட்டும் கேள்.

### Question 1

**Base ஒன்றா?**

✅ ஆம் என்றால்

* Multiply → exponent-களை கூட்டு.
* Divide → exponent-களை கழி.

---

### Question 2

**Exponent ஒன்றா?**

✅ ஆம் என்றால்

* Base-களை பெருக்கு அல்லது வகு.

---

### Question 3

**இரண்டுமே வேறா?**

உதாரணம்:

```text
2³
───
5⁷
```

அப்போது

❌ எந்த basic exponent rule-யும் இல்லை.

அப்படியே விட வேண்டும் அல்லது மதிப்பைக் கணக்கிட வேண்டும்.

## Serge Lang-ன் பார்வை

அவர் rules-ஐ மனப்பாடம் செய்யச் சொல்லவில்லை.

அவர் எப்போதும் முதலில் பார்க்கும் விஷயம்:

> **"எது common?"**

* **Common Base?** → Base rules.
* **Common Exponent?** → Exponent rules.

இதுதான் கணிதவியலாளர்கள் exponent expression-ஐ பார்த்தவுடன் செய்யும் முதல் சிந்தனை.
