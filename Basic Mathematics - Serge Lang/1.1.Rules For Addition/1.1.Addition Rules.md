இந்த §2-ல் ஆசிரியர் உண்மையில் **கூட்டலின் (Addition) விதிகளை** உருவாக்குகிறார். ஆனால் முக்கியமாக, அவர் ஒவ்வொரு விதியும் ஏன் இருக்கிறது என்பதை காட்ட முயற்சிக்கிறார்.

நீ இப்போது `-a` என்பது additive inverse என்று புரிந்துகொண்டதால், இந்த section மிகவும் எளிதாகிவிடும்.

---

# 1. Commutativity

விதி:

a+b=b+a

அர்த்தம்:

> கூட்டும் வரிசை மாறினாலும் பதில் மாறாது.

உதாரணம்:

```text
3 + 5 = 8
5 + 3 = 8
```

Negative உடனும்:

```text
-2 + 5 = 3
5 + (-2) = 3
```

---

## ஏன் இது முக்கியம்?

இதனால்

```text
7 + 1000
```

அல்லது

```text
1000 + 7
```

இரண்டும் ஒன்றே.

கணக்கிடும் சுதந்திரம் கிடைக்கிறது.

---

# 2. Associativity

விதி:

(a+b)+c=a+(b+c)

அர்த்தம்:

> முதலில் எந்த இரண்டு எண்களை கூட்டுகிறோம் என்பது முக்கியமில்லை.

உதாரணம்:

```text
(3 + 5) + 9
= 8 + 9
= 17
```

மற்றொரு பக்கம்:

```text
3 + (5 + 9)
= 3 + 14
= 17
```

---

இதனால் ஆசிரியர் சொல்கிறார்:

```text
3 + 5 + 9
```

என்று parentheses இல்லாமல் எழுதலாம்.

ஏனெனில் எப்படிக் குழுவாக்கினாலும் பதில் ஒன்றே.

---

# 3. N3

ஆசிரியர் ஒரு முக்கியமான முடிவை நிரூபிக்கிறார்.

கொடுக்கப்பட்டது:

```text
a + b = 0
```

அப்போது

```text
b = -a
```

மற்றும்

```text
a = -b
```

---

## ஏன்?

உதாரணம்:

```text
7 + b = 0
```

என்றால்

```text
b = -7
```

தானே?

---

அதேபோல்

```text
-10 + b = 0
```

என்றால்

```text
b = 10
```

---

அதனால்

> 0 வரச்செய்யும் எண் எப்போதும் inverse.

இதைத்தான் N3 சொல்கிறது.

---

# 4. Subtraction-ன் பிறப்பு

இதுவரை subtraction இல்லை.

ஆசிரியர் இப்போது வரையறுக்கிறார்:

```text
a - b
```

என்பது

```text
a + (-b)
```

என்றே அர்த்தம்.

---

உதாரணம்:

```text
10 - 3
```

என்பது

```text
10 + (-3)
```

---

இதுதான் subtraction-ன் உண்மையான definition.

---

# 5. N4

மிக முக்கியம்.

a=-(-a)

---

இப்போது நீ ஏற்கனவே இதைப் புரிந்துகொண்டாய்.

உதாரணம்:

```text
3 = -(-3)
```

ஏன்?

ஏனெனில்

```text
(-3) + 3 = 0
```

அதனால் 3 என்பது -3-ன் inverse.

---

இதேபோல்:

```text
-5 = -(+5)
```

---

# 6. Minus signs குவிந்தால்?

உதாரணம்:

```text
-(-(-3))
```

உள்ளிருந்து தொடங்கு.

முதலில்:

```text
-(-3) = 3
```

பிறகு:

```text
-(3) = -3
```

---

ஆக:

```text
-(-(-3)) = -3
```

---

Rule:

* Even number of minus signs → Positive
* Odd number of minus signs → Negative

---

# 7. N5

இது மிகவும் முக்கியமான rule.

ஆசிரியர் நிரூபிக்கிறார்:

-(a+b)=-a-b

---

இதன் அர்த்தம்:

ஒரு sum-ஐ முழுவதுமாக negative ஆக்கினால்,

ஒவ்வொரு term-ஐயும் negative ஆக்கலாம்.

---

உதாரணம்:

```text
-(3 + 5)
```

=

```text
-3 - 5
```

=

```text
-8
```

---

மற்றொரு உதாரணம்:

```text
-(3 - 7)
```

=

```text
-3 - (-7)
```

---

இப்போது

```text
-(-7)=7
```

---

ஆக:

```text
-3 + 7 = 4
```

---

# 8. Positive + Positive

Rule:

Positive integer + Positive integer

↓

Positive integer

---

உதாரணம்:

```text
17 + 45 = 62
```

இரண்டும் positive.

பதில் positive.

---

# 9. Negative + Negative

உதாரணம்:

```text
-4 + (-7)
```

---

இதையே ஆசிரியர் எழுதுகிறார்:

```text
-(4) + -(7)
```

---

N5 பயன்படுத்தி:

```text
= -(4 + 7)
```

---

```text
= -11
```

---

ஆக இரண்டு negative-களை கூட்டினால் negative வரும்.

---

# 10. Equation Solve செய்வது

உதாரணம்:

```text
x + 3 = 5
```

---

3-ஐ remove செய்ய வேண்டும்.

அதற்காக இருபக்கமும்

```text
-3
```

கூட்டு.

```text
x + 3 - 3 = 5 - 3
```

---

```text
x = 2
```

---

மற்றொரு உதாரணம்:

```text
-2 - y = 5
```

இதன் அர்த்தம்:

```text
-2 + (-y) = 5
```

---

இருபக்கமும் 2 கூட்டு:

```text
-y = 7
```

---

இப்போது inverse எடு:

```text
y = -7
```

---

# இந்த Section-ன் உண்மையான நோக்கம்

ஆசிரியர் சொல்ல விரும்புவது:

1. Addition-க்கு இரண்டு அடிப்படை விதிகள் உள்ளன:

   * Commutativity
   * Associativity

2. Subtraction என்பது தனி operation அல்ல.

   ```text
   a - b
   ```

   என்பது

   ```text
   a + (-b)
   ```

3. `-a` என்பது additive inverse.

4. Equation solve செய்வது என்றால் unwanted term-ன் inverse-ஐ இருபக்கமும் சேர்ப்பது.
