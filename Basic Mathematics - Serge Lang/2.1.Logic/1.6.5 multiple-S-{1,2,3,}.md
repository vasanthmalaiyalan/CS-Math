# முதல் கேள்வி

நாம் வைத்தது:

```text
S = {x | 1 ≤ x ≤ 2}
```

இதன் அர்த்தம்:

**1 முதல் 2 வரை உள்ள எல்லா Real Numbers-ம் S-ல் இருக்கின்றன.**

அதாவது

```text
1
1.1
1.25
1.333...
1.5
1.75
1.9
1.99
2
```

இவ்வளவும் மட்டுமல்ல.

**1 மற்றும் 2-க்கு நடுவில் இருக்கும் முடிவில்லாத (infinitely many) Real Numbers அனைத்தும் S-ல் உள்ளன.**

---

## இப்போது T எப்படி உருவாகிறது?

புத்தகம் சொல்கிறது:

```text
T = {5x | x ∈ S}
```

இதன் அர்த்தம்:

> **S-ல் இருக்கும் ஒவ்வொரு x-யையும் 5-ஆல் பெருக்கு.**

அதனால்

| S-ல் இருக்கும் x | T-ல் கிடைக்கும் 5x |
| ---------------: | -----------------: |
|                1 |                  5 |
|              1.1 |                5.5 |
|              1.2 |                  6 |
|              1.5 |                7.5 |
|              1.9 |                9.5 |
|                2 |                 10 |

ஆக **1 முதல் 2 வரை உள்ள ஒவ்வொரு எண்ணையும்** 5-ஆல் பெருக்க வேண்டும்.

இதனால்தான்

```text
T = {y | 5 ≤ y ≤ 10}
```

என்று வருகிறது.

---

# இரண்டாவது கேள்வி

> `T ⊆ T'` என்றால் என்ன?

முதலில் T' எப்படி வரையறுக்கப்பட்டது என்று பார்ப்போம்.

புத்தகத்தில்

```text
T' = {y | 5 ≤ y ≤ 10}
```

அதாவது

**5 முதல் 10 வரை உள்ள எல்லா Real Numbers.**

---

இப்போது நமக்கு இரண்டு Set-கள் உள்ளன.

### Set T

```text
T = {5x | x∈S}
```

### Set T'

```text
T' = {y | 5≤y≤10}
```

Serge Lang நிரூபிக்க விரும்புவது:

```text
T = T'
```

---

## Step 1

அவர் முதலில் நிரூபிக்கிறார்

```text
T ⊆ T'
```

இதன் அர்த்தம்:

> **T-ல் இருக்கும் ஒவ்வொரு Element-மும் T'-லுமிருக்கிறது.**

---

ஏன்?

Suppose

```text
y ∈ T
```

அப்படியானால்

```text
y = 5x
```

எந்த ஒரு

```text
x ∈ S
```

க்காகவும்.

ஆனால்

```text
x ∈ S
```

என்றால்

```text
1 ≤ x ≤ 2
```

இப்போது முழு inequality-யையும் 5-ஆல் பெருக்கு.

```text
5 ≤ 5x ≤ 10
```

ஆனால்

```text
5x = y
```

அதனால்

```text
5 ≤ y ≤ 10
```

அதாவது

```text
y ∈ T'
```

ஆக

```text
T ⊆ T'
```

நிரூபிக்கப்பட்டது.

---

## ஏன் இன்னும் `T = T'` என்று சொல்லவில்லை?

ஏனெனில்

Subset மட்டும் Equal அல்ல.

உதாரணம்

```text
A = {1,2}
```

```text
B = {1,2,3}
```

இங்கே

```text
A ⊆ B
```

✔

ஆனால்

```text
A = B
```

❌ இல்லை.

---

அதனால் இன்னொரு Direction-ஐயும் நிரூபிக்க வேண்டும்.

```text
T' ⊆ T
```

---

Suppose

```text
y ∈ T'
```

அதாவது

```text
5 ≤ y ≤ 10
```

இப்போது

```text
x = y/5
```

என்று வை.

அப்போது

```text
1 ≤ x ≤ 2
```

அதனால்

```text
x ∈ S
```

மேலும்

```text
y = 5x
```

என்பதால்

```text
y ∈ T
```

ஆக

```text
T' ⊆ T
```

---

இப்போது இரண்டு விஷயங்களும் கிடைத்துவிட்டன.

```text
T ⊆ T'
```

மற்றும்

```text
T' ⊆ T
```

எனவே

```text
T = T'
```

---

## இதை ஒரு படமாகப் பார்

```text
S
[1 ---------------------- 2]
 |  ×5
 ▼
T
[5 ----------------------10]
```

மற்றொரு Set:

```text
T'
[5 ----------------------10]
```

இப்போது Serge Lang நிரூபிப்பது:

> **"×5 செய்து கிடைத்த Set (T) மற்றும் 5 முதல் 10 வரை உள்ள எல்லா எண்களின் Set (T') இரண்டும் உண்மையில் ஒரே Set."**

அதற்காகத்தான் அவர் முதலில் `T ⊆ T'`, பிறகு `T' ⊆ T` என்று இரண்டு திசைகளிலும் நிரூபிக்கிறார். இது Set Theory-யில் **இரண்டு Set-கள் சமம் என்பதை நிரூபிக்கும் அடிப்படை முறை** ஆகும்.
