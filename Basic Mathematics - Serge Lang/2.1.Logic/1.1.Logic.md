இந்த Section (**§2. Logic**) மிகவும் முக்கியமானது.

உண்மையில் Serge Lang இங்கே **Mathematics கற்றுக்கொடுக்கவில்லை.**

அவர் கற்றுக்கொடுப்பது:

> **"Mathematicians எப்படி யோசிக்கிறார்கள்?"**

அதை வரி வரியாகப் பார்ப்போம்.

---

# முதல் Paragraph

அவர் சொல்கிறார்:

> **We always try to keep clearly in mind what we assume and what we prove.**

இதன் அர்த்தம்:

கணிதத்தில் எப்போதும் இரண்டு விஷயங்களை தனித்தனியாகப் பார்க்க வேண்டும்.

1. **Assumption (கொடுக்கப்பட்டது)**
2. **Conclusion (நிரூபிக்க வேண்டியது)**

---

## உதாரணம்

வைத்துக்கொள்.

```text
If x = 5,
then x² = 25.
```

இங்கே

**Assumption**

```text
x = 5
```

**Conclusion**

```text
x² = 25
```

---

Mathematician எப்போதும் கேட்பார்.

> "நான் என்ன Assume செய்கிறேன்?"

பிறகு

> "அதிலிருந்து என்ன prove செய்கிறேன்?"

---

# அடுத்த Sentence

அவர் Proof-ஐ define செய்கிறார்.

> **A proof means a sequence of statements...**

Proof என்றால்

ஒரே ஒரு statement இல்லை.

பல statement-களின் சங்கிலி.

---

உதாரணம்

```text
2x = 6
```

Proof:

```text
2x = 6
```

↓

```text
x = 3
```

↓

```text
அதனால் solution = 3
```

ஒவ்வொரு step-மும்

முந்தைய step-லிருந்து வருகிறது.

இதுதான் Proof.

# Rules of deduction

அவர் சொல்கிறார்

> Deduction rules என்பது Common Sense rules.

உதாரணம்

நான் சொல்கிறேன்

```text
எல்லா மனிதர்களும் இறப்பார்கள்.

ராமன் ஒரு மனிதன்.
```

உடனே

```text
ராமன் இறப்பான்.
```

என்று முடிவு செய்கிறோம்.

இதுதான் Deduction.

# இப்போது முக்கியமான பகுதி

அவர் எழுதுகிறார்

```text
If A,

```

இது Logic-ன் மிகவும் அடிப்படை வடிவம்.

```text

```

இதில்



இதைக் காரணம் → விளைவு (Cause → Effect) போலப் பார்க்கலாம்.

```text
2x=5
```

# இப்போது Converse

அவர் புதிய வார்த்தை அறிமுகப்படுத்துகிறார்.

> Converse

இதன் அர்த்தம்

Sentence-ஐ தலைகீழாக மாற்றுவது.

---

Original

```text
If A,
then B.
```

---

Converse

```text
If B,
then A.
```

---

Example

Original

```text
If 2x=5,
then x=5/2.
```

---

Converse

```text
If x=5/2,
then 2x=5.
```

---

இந்த Example-ல்

Converse-மும் உண்மை.

---

ஆனால்...

இது எப்போதும் உண்மையா?

**இல்லை.**

---

Example

Original

```text
If it rains,
then road is wet.
```

உண்மை.

---

Converse

```text
If road is wet,
then it rained.
```

இது தவறு.

ஏன்?

யாராவது தண்ணீர் ஊற்றியிருக்கலாம்.

---

இதிலிருந்து ஒரு பெரிய பாடம்.

> **ஒரு Statement உண்மையாக இருந்தால், அதன் Converse கட்டாயம் உண்மையாக இருக்க வேண்டிய அவசியமில்லை.**

அதனால்தான் Serge Lang இதை தனியாக அறிமுகப்படுத்துகிறார்.

# If and Only If

இப்போது

Original-ம்

Converse-மும்

இரண்டும் உண்மை.

அப்படியானால்

Mathematician எழுதுவார்

```text
A if and only if B
```

இதற்கு சுருக்கம்

```text
A ⇔ B
```

---

Example

```text
2x=5
```

iff

```text
x=5/2
```

---

இதன் அர்த்தம்

இரண்டு வழியும் உண்மை.

```text
2x=5
```

↓

```text
x=5/2
```

மேலும்

```text
x=5/2
```

↓

```text
2x=5
```

---

இதை diagram-ஆகப் பார்த்தால்

```text
A  ←→  B
```

---

ஆனால்

"If A then B"

என்றால்

```text
A → B
```

ஒரே அம்பு.

---

"If and only if"

என்றால்

```text
A ↔ B
```

இரண்டு அம்பு.

# "Only if"

இங்கே English ஒரு பிரச்சனை தருகிறது.

அவர் சொல்கிறார்

> "A only if B"

இதன் அர்த்தம்

```text
If A,
then B.
```

---

ஆனால்

பலர் தவறாக

```text
If B,
then A.
```

என்று நினைப்பார்கள்.

அதனால் Serge Lang சொல்கிறார்

> **"only if" என்பதை தனியாகப் பயன்படுத்தாதீர்கள்.**

எப்போதும்

```text
A if and only if B
```

என்று முழுவதும் எழுதுங்கள்.

# இந்த Chapter-ன் மிக ஆழமான கருத்து

இந்த Section-ல் Serge Lang கணிதத்தைப் பற்றி பேசவில்லை.

அவர் **கணித மொழி (Mathematical Language)** பற்றி பேசுகிறார்.

அவர் சொல்லும் முக்கியமான நான்கு கருத்துகள்:

1. **Assumption (Given)** — எதை உண்மை என்று ஏற்றுக்கொள்கிறோம்?
2. **Conclusion (To Prove)** — எதை நிரூபிக்க வேண்டும்?
3. **Implication (`If A, then B`)** — A உண்மையானால் B கட்டாயம் உண்மை.
4. **Equivalence (`A if and only if B`)** — A மற்றும் B இரண்டும் ஒன்றுக்கொன்று சமமான நிபந்தனைகள்; A-யிலிருந்து B-க்கும், B-யிலிருந்து A-க்கும் செல்லலாம்.

---

## இதை ஒரு கணிதவியலாளர் எப்படி பார்ப்பார்?

அவர்

```text
If 2x = 5, then x = 5/2.
```

என்றால்,

"இதன் answer 5/2" என்று மட்டும் பார்க்கமாட்டார்.

அவர் மனதில் உடனே இதுபோல் பிரிப்பார்:

```text
Assumption:
2x = 5

        │
        ▼

Logical Rule (divide both sides by 2)

        │
        ▼

Conclusion:
x = 5/2
```

அதாவது **கணிதவியலாளரின் கவனம் answer-ல் இல்லை; answer-க்கு கொண்டு செல்லும் logical chain-ல் இருக்கிறது.** அதனால்தான் இந்த Chapter-க்கு "Logic" என்று பெயர் வைத்துள்ளார்.
