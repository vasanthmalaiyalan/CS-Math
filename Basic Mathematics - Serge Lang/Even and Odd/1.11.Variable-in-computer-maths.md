## Programming Variable vs Math Variable

### Mathematics

```text
x + 5 = 10
```

இங்கே:

```text
x
```

ஒரு தெரியாத எண்.

---

### C++

```cpp
int x = 5;
```

இங்கே:

```cpp
x
```

memory-யில் ஒரு value-ஐ வைத்திருக்கும் பெயர்.

---

இரண்டிற்கும் தொடர்பு இருக்கிறது.

Programming variable என்ற concept-ஏ Mathematics variable-லிருந்து வந்தது.

இவை எல்லாம் Programming variables அல்ல.

இவை **Mathematical variables** — அதாவது தெரியாத அல்லது பொதுவான (general) எண்களை குறிக்கும் symbols.

உண்மையில் Algebra-வின் பெரும்பகுதி முழுவதும் variables-ஐ வைத்துதான் கட்டப்பட்டுள்ளது. Variables இல்லாமல் "எல்லா even numbers-க்கும் இது உண்மை" போன்ற statements-ஐ எழுதவே முடியாது.
