ஆம். இதுவரை நீ புரிந்துகொள்ள முயற்சித்த முக்கிய கருத்துகள்:

1. `-a` = additive inverse
2. `a + (-a) = 0`
3. `-(-a) = a`
4. Commutativity
5. Associativity
6. `a - b = a + (-b)`

இவற்றை **ஒரே C++ program-ல்** பார்க்கலாம்.

நீ code-ஐ type செய்து, output-ஐப் பார்த்து, ஒவ்வொரு line-ஐ கைமுறையாக verify செய்ய வேண்டும்.

```cpp
#include <iostream>

int main()
{
    int a = 5;
    int b = 3;

    std::cout << "a = " << a << '\n';
    std::cout << "b = " << b << '\n';

    std::cout << "\n=== Additive Inverse ===\n";

    std::cout << "a      = " << a << '\n';
    std::cout << "-a     = " << -a << '\n';
    std::cout << "a+(-a) = " << a + (-a) << '\n';

    std::cout << "\n=== Double Minus ===\n";

    std::cout << "-(-a) = " << -(-a) << '\n';

    std::cout << "\n=== Subtraction ===\n";

    std::cout << "a-b           = " << a - b << '\n';
    std::cout << "a+(-b)        = " << a + (-b) << '\n';

    std::cout << "\n=== Commutativity ===\n";

    std::cout << "a+b = " << a + b << '\n';
    std::cout << "b+a = " << b + a << '\n';

    std::cout << "\n=== Associativity ===\n";

    int c = 9;

    std::cout << "(a+b)+c = "
              << (a + b) + c
              << '\n';

    std::cout << "a+(b+c) = "
              << a + (b + c)
              << '\n';

    std::cout << "\n=== Negative Of A Sum ===\n";

    std::cout << "-(a+b)      = "
              << -(a + b)
              << '\n';

    std::cout << "-a + (-b)   = "
              << (-a) + (-b)
              << '\n';

    return 0;
}
```

---

## இதை Run செய்த பிறகு

Output:

```text

=== Additive Inverse ===

-a     = -5

=== Double Minus ===

-(-a) = 5

=== Subtraction ===

a-b           = 2
a+(-b)        = 2

=== Commutativity ===

a+b = 8
b+a = 8

=== Associativity ===

(a+b)+c = 17
a+(b+c) = 17

=== Negative Of A Sum ===

-(a+b)      = -8
-a + (-b)   = -8
```
