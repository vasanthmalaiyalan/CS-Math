ஆனால் அவர் இதை இப்படி பார்க்கிறார்:

```text
-(-5)
```

=

> **the additive inverse of -5**

அதாவது:

> **"-5-ன் additive inverse"**

# Scenario 1: Account Balance

உன்னிடம் ₹5 இருக்கிறது.

```text
Balance = +5
```

---

இப்போது ₹5 அபராதம் (Fine) வந்தது.

```text
-5
```

இங்கே:

```text
-5
```

என்பது

> +5-ஐ cancel செய்யும் entry

அதாவது:

```text
5 + (-5) = 0
```

# Scenario 2: Fine தவறாக வந்தது

System தவறாக ₹5 அபராதம் போட்டுவிட்டது.

அதை cancel செய்ய வேண்டும்.

---

Original:

```text
-5
```

---

Cancel Entry:

```text
-(-5)
```

---

ஏன்?

ஏனெனில்:

```text
(-5) + (-(-5)) = 0
```

---

Result:

```text
-(-5) = 5
```

---

Database:

| Transaction  | Value |
| ------------ | ----- |
| Wrong Fine   | -5    |
| Reverse Fine | +5    |

Net:

```text
-5 + 5 = 0
```
