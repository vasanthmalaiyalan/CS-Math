# 2. "5" என்றால் உண்மையில் என்ன?

அதாவது

"5" என்பது ஒரு பொருள் அல்ல.

அதன் behavior தான் முக்கியம்.

```text

```

ஆனால் algebra-வில்

> எந்த எண்ணுடன் கூட்டினாலும் அதே எண்ணை விட்டுவிடும் ஒன்று.

அதாவது

```text

100+0=100
```

எனவே

```text

```
