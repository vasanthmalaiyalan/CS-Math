இது *Arithmetic* புத்தகத்தின் மிகவும் அழகான chapter-களில் ஒன்று. Paul Lockhart இங்கே "fraction என்றால் என்ன?" என்பதை பள்ளியில் கற்பிப்பது போல விதிமுறைகளாக அல்ல, மனிதர்கள் ஏன் fraction-ஐ கண்டுபிடித்தார்கள் என்ற கதையாக விளக்குகிறார்.

இந்த chapter-ன் முக்கிய கருத்துகளை பகுதிகளாகப் பார்ப்போம்.

# 2. Division-ல் leftover (remainder)

17 jelly beans

```

```

```

```

```

```

bean-ஐ வெட்டலாம்.

# 16. Reciprocal

2 மற்றும்

1/2

ஒருவருக்கொருவர் opposite.

ஏன்?

```
2 × 1/2 =1
```

அவை ஒருவரை ஒருவர் cancel செய்கின்றன.

இதுவே reciprocal.

---

Example

```
2/3
```

reciprocal

```
3/2
```

ஏனெனில்

```
2/3 × 3/2 =1
```
