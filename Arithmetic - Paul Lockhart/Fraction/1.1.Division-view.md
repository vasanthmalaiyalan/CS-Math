இதனால் Fraction chapter உண்மையில்

**"17÷4 = 4¼"**

என்ற chapter அல்ல.

அது

> **"உலகம் தொடர்ச்சியானதா அல்லது துண்டுகளால் ஆனதா?"**

என்ற மிகவும் ஆழமான mathematical மற்றும் philosophical கேள்விக்கான கதவாக இருக்கிறது.

இதுதான் Paul Lockhart இந்த chapter-ஐ தொடங்கும் போது வாசகரை மெதுவாக அழைத்துச் செல்லும் இடம்.

------------------------------------------------

``Fraction என்பது "மீதத்தை வெட்டுவது"
அந்த 1 jelly bean-ஐ
4 சம பாகங்களாக வெட்டுகிறோம்.
அப்பொழுது
ஒவ்வொருவருக்கும்
Plain text
4 + 1/4
இதுதான் fraction.
Fraction`, நீ கூறியபடி இது எப்படி சாத்தியம் plus sign இதில் போட்டுள்ளதால் `4 + 1/4`distribution எப்படி இரு பக்கமும் நடக்குமா? மற்றும் remaining ஒன்றை நாம் எப்படி 1/4 இப்படிப் பிடிக்க முடியும்? என்ன வேல்யூ கிடைத்ததோ like if four என்றால் 1/4 என்று பிரிக்கிறாயா`


மிகச் சிறந்த கேள்வி. இதுதான் fraction-ன் இதயம். பள்ளியில் பலர் இங்கேயே குழப்பமடைகிறார்கள், ஏனெனில் அவர்கள் `4 + 1/4` என்பதை ஒரு symbol ஆகப் பார்க்கிறார்கள்; ஆனால் Lockhart அதை **பகிர்தல் (sharing)** என்ற செயலாகப் பார்க்கிறார்.

## 17 jelly beans, 4 பேர்

முதலில் முழு jelly beans-ஐப் பகிர்வோம்.

```
17 = 4 + 4 + 4 + 4 + 1
```

அதாவது

```
Person A → 4
Person B → 4
Person C → 4
Person D → 4

Remaining → 1 jelly bean
```

இதுவரை எல்லோருக்கும் 4 கிடைத்துவிட்டது.

## மீதமுள்ள 1 jelly bean என்ன செய்வது?

ஏனெனில் இன்னும் 4 பேருக்கும் சமமாக கொடுக்க வேண்டும்.

```
Person B → 1/4
Person C → 1/4
Person D → 1/4
```

## Distribution இரு பக்கமும் நடக்கிறதா?

> "four என்பதால் 1/4 என்று பிரிக்கிறாயா?"

ஆம்.

ஏனெனில்

**மீதமுள்ள ஒரு முழு பொருளை நான்கு சமமான பாகங்களாக வெட்டுகிறோம்.**

அதனால்

```

```

என்பது உண்மை.

---

இதை இன்னும் Lockhart பாணியில் எழுதினால்:

```
17
= 16 + 1
= 4×4 + 1
= 4×4 + 4×(1/4)
= 4×(4 + 1/4)
```

ஆகவே

```
17 ÷ 4 = 4 + 1/4
```

அதாவது

```
17 ÷ 4 = 4¼
```

இங்கே fraction என்பது ஒரு புதிய எண் அல்ல.

**"மீதமுள்ள 1-ஐ, பகிர வேண்டிய மக்களின் எண்ணிக்கைக்கு ஏற்ப சமமான பாகங்களாக வெட்டியதன் விளைவு"** தான் `1/4`. இதுதான் fraction-ன் பிறப்பு.
