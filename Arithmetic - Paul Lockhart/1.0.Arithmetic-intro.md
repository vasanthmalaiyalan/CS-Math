# 1. உலகம் முழுக்க "Things" தான்

நம்மைச் சுற்றி என்ன இருக்கு?

* மரங்கள் 🌳
* கற்கள் 🪨
* மிருகங்கள் 🐄
* நட்சத்திரங்கள் ⭐
* மணல் துகள்கள்

எல்லாமே "things".

இந்த things-ஐ நாம் சில நேரங்களில் count பண்ணுறோம்.

ஆனா author ஒரு interesting question கேக்குறார்:

> "நாம ஏன் count பண்ணுறோம்?"

# 2. உண்மையில் நாம் அதிகம் count பண்ணவே மாட்டோம்

உதாரணம்:

உங்க வீட்டுல அரிசி இருக்கு.

நீங்க தினமும்:

> "அதில் 12,437 தானியங்கள் இருக்கா?" 🤔

என்று எண்ண மாட்டீங்க.

அதற்கு பதில்:

> "போதும்."

என்று நினைப்பீங்க.

அதாவது பெரும்பாலான நேரம் exact number தேவையில்லை.

"நிறைய இருக்கு"

அல்லது

"குறைவா இருக்கு"

அப்படின்னு தெரிஞ்சா போதும்.

# 3. Count பண்ணுறதுக்கான முக்கிய காரணம்

Author சொல்றார்:

> **Counting is really about comparison.**

அதாவது

**Count பண்ணுறது = Compare பண்ணுறதுக்காக.**

உதாரணம்:

நீங்க pocket-ல ₹20 வைச்சிருக்கீங்க.

பொருள் விலை ₹16.

நீங்க compare பண்ணுறீங்க:

₹20 > ₹16

அதனால் வாங்க முடியும்.
