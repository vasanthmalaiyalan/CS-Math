# 7. Division என்பது sharing

248 jelly beans

7 நண்பர்கள்

ஒவ்வொருவருக்கும் சமமாகப் பகிர வேண்டும்.

ஆனால் ஒவ்வொருவருக்கும் எவ்வளவு கிடைக்கும் என்பது தெரியாது.

அந்த unknown quantity x என வைத்தால்

```
x+x+x+x+x+x+x =248
```

அதாவது

```
7×x=248
```

Division என்றால்

> Multiplication-ல் தெரியாததை கண்டுபிடிப்பது.

---

# 8. Division என்பது "undo multiplication"

Multiplication:

```
35×7=245
```

Division:

```
245÷7=35
```

அல்லது

```
245÷35=7
```

ஒரு செயலை செய்தால்

அதை reverse செய்ய division வருகிறது.

# 11. Equality தான் முக்கியம்

7 நண்பர்கள் இருக்கிறார்கள்.

248 jelly beans.

ஒருவருக்கு 40

என்றால் சண்டை வரும்.

அதனால் division-ன் அடிப்படை நோக்கம்:

> Equal sharing.

# 12. Remainder ஏன் வருகிறது?

248÷7

7×35=245

மீதம் 3

அதாவது

```
248 = 35×7 +3
```

அந்த 3-ஐ சமமாகப் பகிர முடியாது.

அதனால் remainder.

Division எல்லா நேரமும் perfect ஆக வராது.

இதுவே Euclidean division.

```
Dividend = Divisor×Quotient + Remainder
```
