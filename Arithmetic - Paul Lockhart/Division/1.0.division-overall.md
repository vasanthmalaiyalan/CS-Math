# இந்த chapter-ன் உண்மையான message

Paul Lockhart சொல்ல வருவது:

> Division என்பது long division algorithm அல்ல.

Division என்பது

**"எனக்கு total தெரியும்; எத்தனை equal groups வேண்டும் என்பதும் தெரியும்; ஒவ்வொரு group-லும் எவ்வளவு இருக்க வேண்டும்?"**

என்ற கேள்வி.

அதாவது

```
Multiplication :
known quantity → repeated copies → total

Division :
total + number of copies → unknown quantity
```

இவை ஒன்றுக்கொன்று inverse operations.

மேலும்,

Long division என்பது வெறும்

> estimate → multiply → subtract → repeat

என்ற மனிதர்கள் உருவாக்கிய ஒரு practical bookkeeping method மட்டுமே.

அதுதான் இந்த chapter-ன் மைய கருத்து.
