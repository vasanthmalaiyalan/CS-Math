# Deepest Idea in the Chapter

Lockhart is secretly preparing you for our modern number system.

He wants you to notice:

### Egyptian System

Value comes from symbol.

---

### Roman System

Mostly value comes from symbol.

But the abacus starts introducing value from position.

---

### Modern Decimal System

Value comes almost entirely from position.

---

So the journey is:

```text
Rocks
↓
Egyptian symbols
↓
Roman symbols
↓
Roman abacus
↓
Place value
↓
Modern numbers (0-9)
```

This is why this chapter is important. It's not really about Romans. It's about showing how humans slowly discovered the incredibly powerful idea that:

> **A symbol's value can depend on where it is placed.**

That single idea eventually leads to writing huge numbers, fast arithmetic, algebra, computers, and programming languages. 🚀
