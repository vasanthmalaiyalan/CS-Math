# 1. 5×8 என்றால் என்ன?

அதன் அர்த்தம்:

```text
8 + 8 + 8 + 8 + 8
```

அல்லது

5 copies of 8.

இதனை rocks ஆகப் பார்த்தால்:

```text
oooooooo
oooooooo
oooooooo
oooooooo
oooooooo
```

5 rows, ஒவ்வொரு row-ல் 8 rocks.

அதாவது:

```text
5×8
```

ஆனால் இதையே columns ஆகப் பார்த்தால்:

```text
o o o o o o o o
o o o o o o o o
o o o o o o o o
o o o o o o o o
o o o o o o o o
```

8 columns, ஒவ்வொரு column-ல் 5 rocks.

அதாவது:

```text
8×5
```

ஆனால் rocks மாறவில்லை.

அதனால்

```text
5×8 = 8×5
```

இதைத்தான் Lockhart "symmetry" என்கிறார்.

# 2. 17×2 ஏன் 2×17 ஆக மாற்றலாம்?

நேரடியாக:

```text
2+2+2+2+2+...
```

17 முறை சேர்க்கலாம்.

ஆனால் rectangle symmetry காரணமாக:

```text
17×2 = 2×17
```

அதாவது,

17-ஐ double செய்தால் போதும்.

```text
17+17=34
```

இது எளிது.

# 5. இன்னொரு clever way

6×8

நேரடியாக:

```text
8+8+8+8+8+8
```

செய்ய வேண்டாம்.

Lockhart சொல்கிறார்:

முதலில் double:

```text
8→16
```

மீண்டும் 8 சேர்த்து:

```text
16+8=24
```

இது 3 copies.

ஆனால் 6 copies வேண்டுமல்லவா?

24-ஐ double செய்:

```text
24+24=48
```

அவ்வளவு தான்.
