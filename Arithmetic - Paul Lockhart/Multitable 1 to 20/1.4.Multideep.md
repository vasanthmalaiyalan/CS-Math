# Lockhart சொல்லும் மிகப் பெரிய கருத்து

Multiplication என்பது

> repeated addition அல்ல.

அது

> breaking, rearranging, grouping, distributing.

அதாவது

```
(a+b)(c+d)
```

என்பது formula அல்ல.

ஒரு பெரிய rectangle-ஐ

சிறிய rectangles-ஆக வெட்டிப் பார்த்ததன் விளைவு.

இதனால்தான் அவர் சொல்கிறார்:

> "The product of two sums is equal to the sum of the individual products."

இது arithmetic-இல் மட்டுமல்ல,

பின்னர் algebra, matrices, vectors, polynomials, calculus ஆகிய அனைத்திற்கும் அடிப்படையான மிக முக்கியமான property ஆக மாறுகிறது.

இந்தப் பகுதி முழுவதும் **"ஏன் Hindu-Arabic place value system உலகையே வென்றது?"** என்பதற்கான பதில். இதை மெதுவாக ஆழமாகப் பார்ப்போம்.

# 1. Egyptian system-ல் ×10 என்பது என்ன?

Egyptians-க்கு ஒவ்வொரு place-க்கும் தனி symbol இருந்தது.

உதாரணம்:

* 1 → ஒன்று
* g → 10
* p → 100
* X → 1000

இப்போது ஒரு எண்ணை எடுத்துக்கொள்வோம்.

```
q j 3
```

(சரியான symbol-களைப் பற்றி கவலைப்பட வேண்டாம்.)

×10 செய்ய வேண்டும்.

சாதாரணமாக:

```
N + N + N + ... (10 times)
```

செய்யலாம்.

ஆனால் Egyptian-க்கு ஒரு clever trick உள்ளது.

### 10 ones = 1 ten

அதனால்

```
1 → g
g → p
p → X
```

அதாவது எல்லா symbols-உம் ஒரு level மேலே upgrade ஆகிவிடும்.

# 2. Roman system-லும் அதே idea

Roman:

```
I = 1
X = 10
C =100
M =1000
```

×10 செய்தால்

```
I → X
V → L
X → C
C → M
```
